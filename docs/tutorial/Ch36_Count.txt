Chapter 36: Count Pack

The count pack (module count, prefix cn_) provides fourteen predicates for
counting cells, colors, and regions in grids and region lists. Layer 54,
requires grid.

Load the pack:

    :- use_module(library(count)).


Counting Cells by Color

count_color_count/3 counts how many cells in a grid equal a specific color.

    ?- count_color_count([[0,1,0],[1,0,1]], 0, N).
    N = 3.

count_by_value/3 counts occurrences of a value in a flat list.

    ?- count_by_value([1,2,1,3,1], 1, N).
    N = 3.


Color Histogram

count_histogram/3 produces parallel lists of all colors and their cell counts,
sorted by color value ascending.

    ?- count_histogram([[0,1,2],[0,1,2]], Colors, Counts).
    Colors = [0,1,2], Counts = [2,2,2].

    ?- count_histogram([[0,0,0],[0,1,0],[0,0,0]], Colors, Counts).
    Colors = [0,1], Counts = [8,1].


Most and Least Frequent Colors

count_max_color/2 returns the most frequently occurring color. When tied, it
returns the lowest color value.

count_min_color/2 returns the least frequently occurring color.

    ?- count_max_color([[0,0,0],[0,1,0],[0,0,0]], C).
    C = 0.   % 0 appears 8 times, 1 appears once.

    ?- count_min_color([[0,0,0],[0,1,0],[0,0,0]], C).
    C = 1.   % 1 appears only once.


Row and Column Color Presence

count_color_rows/3 counts rows containing a specific color at least once.
count_color_cols/3 counts columns containing a specific color at least once.

    ?- count_color_rows([[0,1,2],[0,1,2],[9,9,9]], 0, N).
    N = 2.   % color 0 in rows 0 and 1, not row 2.

    ?- count_color_cols([[0,1,0],[0,1,0]], 0, N).
    N = 2.   % color 0 in cols 0 and 2.


Value Diversity in Rows and Columns

count_row_distinct/3 counts distinct values in a specific row.
count_col_distinct/3 counts distinct values in a specific column.

    ?- count_row_distinct([[1,2,3],[4,4,4]], 0, N).
    N = 3.   % row 0 has three distinct values.

    ?- count_col_distinct([[1,2],[1,3],[1,4]], 0, N).
    N = 1.   % column 0 has all 1s.


Grid Size

count_grid_total/2 returns the total cell count (Rows * Cols).

    ?- count_grid_total([[1,2,3],[4,5,6]], N).
    N = 6.


Comparing Two Grids

count_equal_cells/3 counts positions where two same-size grids have equal values.
count_diff_cells/3 counts positions where they differ.

    ?- count_equal_cells([[1,2],[3,4]], [[1,9],[3,4]], N).
    N = 3.   % only position (0,1) differs.

    ?- count_diff_cells([[1,2],[3,4]], [[1,9],[3,4]], N).
    N = 1.


Region Colors

count_region_color/3 returns the color of a region by looking up the grid value
at the region's first cell. For a connected component, all cells share the
same color, so this gives the region's color.

    ?- count_region_color([[0,1],[2,3]], [r(0,1), r(1,1)], C).
    C = 1.

count_regions_per_color/3 tallies regions by color, producing Color-Count pairs.

    ?- Regions = [[r(0,0)], [r(0,1)], [r(1,0)]],
       count_regions_per_color([[0,1],[0,2]], Regions, Pairs).
    Pairs = [0-2, 1-1].  % two color-0 regions, one color-1 region.


Exported Predicates

count_color_count/3       - cell count for a specific color
count_histogram/3         - sorted color histogram
count_max_color/2         - most frequent color
count_min_color/2         - least frequent color
count_color_rows/3        - rows containing a color
count_color_cols/3        - columns containing a color
count_row_distinct/3      - distinct values in a row
count_col_distinct/3      - distinct values in a column
count_grid_total/2        - total cell count
count_equal_cells/3       - matching cell count between two grids
count_diff_cells/3        - differing cell count between two grids
count_region_color/3      - color of a region from the grid
count_regions_per_color/3 - per-color region tally
count_by_value/3          - value count in a flat list
