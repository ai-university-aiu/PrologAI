Chapter 191: gridcolor Pack - Color Analysis, Recoloring, Histogram, Threshold, and Dominant Color (gc_*, Layer 209)

Overview

The gridcolor pack (Layer 209, gc_* prefix) provides fourteen predicates for
color analysis and transformation: counting, histograms, frequency ranking,
unique color listing, cell lookup, recoloring, color mapping, binary
thresholding, dominant color detection, fraction computation, and frequency-
sorted color lists.

Load the pack:

:- use_module(pack(gridcolor/prolog/gridcolor)).


1. Counting and Analysis

gridcolor_count(+Grid, +Color, -N)

Count cells in Grid equal to Color.

Example:
G = [[r,r,x],[r,x,b],[x,x,r]].
gridcolor_count(G, r, N).
% N = 4

gridcolor_histogram(+Grid, -Pairs)

Pairs is a Color-Count list for every distinct color, sorted alphabetically.

Example:
gridcolor_histogram([[r,x],[r,r]], Pairs).
% Pairs = [r-3, x-1]

gridcolor_most_frequent(+Grid, -Color)

Color is the most frequently occurring color.

gridcolor_least_frequent(+Grid, -Color)

Color is the least frequently occurring color.

gridcolor_unique_colors(+Grid, -Colors)

Colors is the sorted list of distinct color atoms present.

gridcolor_count_colors(+Grid, -N)

N is the number of distinct colors.

gridcolor_sorted_by_freq(+Grid, -Colors)

Colors sorted most-frequent-first.

Example:
G = [[r,r,r],[r,x,x],[x,x,b]].
gridcolor_sorted_by_freq(G, Colors).
% Colors = [r, x, b]  (r:4, x:4, b:1 -- ties broken alphabetically... actually by sort order)


2. Cell Lookup

gridcolor_color_cells(+Grid, +Color, -Cells)

Cells is the list of R-C positions (row-major order) where Grid[R][C] = Color.

Example:
gridcolor_color_cells([[r,x],[x,r]], r, Cells).
% Cells = [0-0, 1-1]

gridcolor_contains(+Grid, +Color)

Succeed if Grid has at least one cell of Color.


3. Color Transformation

gridcolor_recolor(+Grid, +OldColor, +NewColor, -Result)

Replace every occurrence of OldColor with NewColor. Other cells unchanged.

Example:
gridcolor_recolor([[r,b],[b,r]], r, x, R).
% R = [[x,b],[b,x]]

gridcolor_color_map(+Grid, +Map, -Result)

Apply a color mapping Map (list of Old-New pairs). Unmapped colors pass through.

Example:
gridcolor_color_map([[r,g],[g,r]], [r-b, g-r], R).
% R = [[b,r],[r,b]]

gridcolor_threshold(+Grid, +Colors, +FgColor, +BgColor, -Result)

Binary grid: cells in Colors become FgColor; all others become BgColor.

Example:
gridcolor_threshold([[r,x,b],[r,r,x]], [r,b], sel, bg, R).
% R = [[sel,bg,sel],[sel,sel,bg]]


4. Statistics

gridcolor_dominant(+Grid, +BgColor, -Color)

Most frequent non-BgColor color. Fails if Grid has only BgColor cells.

Example:
gridcolor_dominant([[x,r,x],[x,r,r]], x, Color).
% Color = r

gridcolor_fraction(+Grid, +Color, -Frac)

Fraction [0.0..1.0] of cells matching Color.

Example:
gridcolor_fraction([[r,r,r],[r,r,r],[r,r,r]], r, F).
% F = 1.0


5. Summary of Predicates

gridcolor_count/3          - count cells matching Color
gridcolor_histogram/2      - Color-Count pairs sorted by Color name
gridcolor_most_frequent/2  - most common color
gridcolor_least_frequent/2 - least common color
gridcolor_unique_colors/2  - sorted list of distinct colors
gridcolor_color_cells/3    - R-C list of cells matching Color
gridcolor_contains/2       - succeed if Color present in Grid
gridcolor_recolor/4        - replace OldColor with NewColor throughout
gridcolor_color_map/3      - apply list of Old-New mappings
gridcolor_threshold/5      - binary grid: Colors list -> FgColor; others -> BgColor
gridcolor_dominant/3       - most frequent non-BgColor
gridcolor_fraction/3       - float fraction of cells matching Color
gridcolor_count_colors/2   - number of distinct colors present
gridcolor_sorted_by_freq/2 - colors sorted most-frequent-first
