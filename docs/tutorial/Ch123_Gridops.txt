Chapter 123: Grid Collection Operations Pack (Layer 141, go_* prefix)

The gridops pack provides 14 predicates for reasoning across a collection of
multiple 2D integer grids simultaneously: position-set queries (always, never,
sometimes), per-cell value counting, modal value per cell, stable and unstable
cell detection, grid equality testing, and elementwise arithmetic operations
(add, subtract, max, min, overlay, intersect).

Predicates:

gridops_always(+Grids, +V, -Cells)
  Cells is the sorted list of R-C positions where EVERY grid has value V.

gridops_never(+Grids, +V, -Cells)
  Cells is the sorted list of R-C positions where NO grid has value V.

gridops_sometimes(+Grids, +V, -Cells)
  Cells is the sorted list of R-C positions where SOME (>=1) grid has value V
  but NOT all grids do. Deduplicated via sort.

gridops_count_v(+Grids, +V, -CountGrid)
  CountGrid is a grid of integers; each cell (R,C) holds the count of how
  many grids in Grids have value V at (R,C).

gridops_modal(+Grids, -ModalGrid)
  ModalGrid is a grid; each cell (R,C) holds the most frequent value across
  all grids at that position. Ties broken by smallest value.

gridops_stable(+Grids, -Triples)
  Triples is the list of R-C-V terms for positions where ALL grids agree on
  value V. V comes from the first grid; verified against the rest.

gridops_unstable(+Grids, -Cells)
  Cells is the sorted list of R-C positions where the grids disagree
  (not all have the same value).

gridops_eq(+G1, +G2)
  Succeeds if G1 and G2 are cell-for-cell identical (structural unification).

gridops_add(+G1, +G2, -G)
  G[R][C] = G1[R][C] + G2[R][C]. Grids must have identical dimensions.

gridops_sub(+G1, +G2, -G)
  G[R][C] = G1[R][C] - G2[R][C].

gridops_emax(+G1, +G2, -G)
  G[R][C] = max(G1[R][C], G2[R][C]).

gridops_emin(+G1, +G2, -G)
  G[R][C] = min(G1[R][C], G2[R][C]).

gridops_overlay(+Grids, +Bg, -G)
  G[R][C] = first non-Bg value at (R,C) scanning Grids in order; Bg if all
  grids have Bg at (R,C).

gridops_intersect(+Grids, +Bg, -G)
  G[R][C] = V if ALL grids have the same non-Bg value V at (R,C); else Bg.

Usage examples:

Gs = [[[1,2],[3,0]], [[1,5],[3,0]]].  % two 2x2 grids
gridops_always(Gs, 1, C).    % C=[0-0]: only (0,0) has 1 in both
gridops_always(Gs, 0, C).    % C=[1-1]: only (1,1) has 0 in both
gridops_never(Gs, 0, C).     % C=[0-0,0-1,1-0]: these positions never have 0
gridops_sometimes(Gs, 2, C). % C=[0-1]: (0,1) has 2 in Gs[0] but 5 in Gs[1]
gridops_count_v(Gs, 0, CG).  % CG=[[0,0],[0,2]]
gridops_modal(Gs, MG).       % MG=[[1,2],[3,0]]; tie at (0,1) won by 2 (smallest)
gridops_stable(Gs, T).       % T=[0-0-1,1-0-3,1-1-0]
gridops_unstable(Gs, C).     % C=[0-1]

A = [[1,2],[3,4]]. B = [[10,20],[30,40]].
gridops_add(A, B, G).   % G=[[11,22],[33,44]]
gridops_sub(B, A, G).   % G=[[9,18],[27,36]]
gridops_emax(A, B, G).  % G=[[10,20],[30,40]]
gridops_emin(A, B, G).  % G=[[1,2],[3,4]]

OvGs = [[[1,0],[0,2]], [[0,3],[4,0]]].  % Bg=0
gridops_overlay(OvGs, 0, G). % G=[[1,3],[4,2]]: first non-0 per cell
gridops_intersect(OvGs, 0, G). % G=[[0,0],[0,0]]: no cell has same non-0 in both

gridops_eq([[1,2],[3,4]], [[1,2],[3,4]]). % succeeds
gridops_eq([[1,2],[3,4]], [[1,2],[3,5]]). % fails

Key implementation notes:

gridops_always uses forall(member(Grid, Grids), (nth0(R, Grid, Row), nth0(C, Row, V)))
  which checks every grid in the list. forall/2 is a SWI-Prolog built-in.

gridops_sometimes uses \+ forall(...) to test "not all grids have V". The findall
  can produce duplicate R-C pairs (one per grid that has V), so sort removes them.

gridops_count_v uses nested findall: outer by row, inner by column, innermost by grid.
  The innermost findall collects one element per grid having V; length gives count.

gridops_modal uses gridops_mode_/2 which computes the mode via sort+include+msort+last.
  For ties, msort on N-V pairs sorts same-count values ascending, so the
  first candidate (smallest V) is the modal value.

gridops_stable tests agreement by checking the first grid's value against all others
  using forall. This is O(H*W*|Grids|) overall.

gridops_overlay uses if-then-else (->): member backtracking finds the first non-Bg
  grid; the soft-cut (implied by ->) stops at the first match.

gridops_intersect checks whether the first grid has non-Bg and whether all other
  grids match. The whole check is wrapped in if-then-else.

gridops_add, gridops_sub, gridops_emax, gridops_emin use maplist/4 with named helper predicates
  for row-level and cell-level operations.

Built-ins: sort/2, msort/2, length/2, between/3, forall/2, findall/3.
