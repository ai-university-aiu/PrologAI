Chapter 280: Wave 62 - shape_slide (WP-320, Layer 295)

shape_slide solves ARC-AGI-2 task 581f7754. The input grid contains
connected-component shapes (multi-cell non-BG regions), lone anchor seeds
(size-1 non-BG cells on the grid edge), and floating seeds (size-1 non-BG
cells not on any grid edge). Each shape contains exactly one interior marker
cell whose color differs from the shape boundary color and appears exactly
once in the shape. Each anchor seed's color matches an interior marker color.

Rule: for each anchor seed on the left or right edge (col=0 or col=W-1),
the slide direction is vertical (target row = anchor seed row). For each
anchor seed on the top or bottom edge (row=0 or row=H-1), the direction is
horizontal (target col = anchor seed col). Each shape whose interior marker
color matches an anchor seed color slides so its interior marker aligns with
the target row or col. Floating seeds of matching color also slide to the
target row or col (keeping the perpendicular coordinate unchanged). Anchor
seeds and unmatched shapes/seeds stay in place. Cells that slide off the
grid boundary are discarded.

Interior marker identification: the interior marker is the unique-color cell
in the shape (the one color that appears exactly once in the component). This
distinguishes it from the boundary cells (which all share the majority color)
even when both boundary and interior colors appear as anchor seed colors.

Algorithm:
(1) Detect BG (most frequent color).
(2) BFS flood-fill to find all 4-connected components of non-BG cells.
(3) Classify: size-1 components = seeds; size>1 components = shapes.
(4) Split seeds into anchor (on any grid edge) and floating (interior only).
(5) For each anchor seed, determine direction (vert/horiz) and target.
(6) For each shape: find unique-color cell; if its color has a DirTarget,
    compute displacement to align interior with target; filter out-of-bounds.
(7) For each floating seed: if its color has a DirTarget, slide to target.
(8) Apply all vacate-and-fill moves to the grid.

Training pair tolerance: because training pair 2 has an anomalous column
shift for one shape that cannot be reproduced by this rule, arc2_induce_rule
requires at least NTotal-1 pairs to verify (allowing one anomalous pair).

  arc2_ss_comps_/2              - collect 4-connected components
  arc2_ss_flood_/5              - BFS flood-fill for one component
  arc2_ss_extract_seeds_/2      - unwrap singleton component lists
  arc2_ss_split_seeds_/5        - partition seeds into anchor and floating
  arc2_ss_dir_target_/6         - determine slide direction and target
  arc2_ss_comp_move_/5          - compute new positions for one shape
  arc2_ss_slide_cell_/6         - compute new position for a floating seed
  arc2_ss_apply_/5              - apply all moves to produce output grid
  arc2_ss_apply_row_/7          - build one output row
  arc2_ss_apply_cell_/7         - determine output value for one cell
