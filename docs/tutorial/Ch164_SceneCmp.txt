Overview

The scenecmp pack (Layer 182) compares two lists of obj(Color, Cells) terms at the
inventory level. Given a Before scene and an After scene, it answers: did the number
of objects change? Did new colors appear? Did new shapes appear?

The pack does not require an explicit object-to-object matching. It compares global
inventories: the color set, the form set, and object and cell counts.

The pack exports 14 predicates under the sm_* prefix.


Predicate Reference

The predicates fall into three groups: introspection (single scene), symmetric
comparison (same question for both), and asymmetric comparison (what changed).


Introspection Predicates

scene_compare_n_objs(+Objs, -N)
    Count of objects in the list.

    ?- scene_compare_n_objs([obj(r,[r(0,0)]), obj(b,[r(0,1)])], N).
    N = 2.

scene_compare_total_cells(+Objs, -N)
    Total cell count across all objects.

    ?- scene_compare_total_cells([obj(r,[r(0,0)]), obj(r,[r(0,0),r(0,1),r(0,2)])], N).
    N = 4.

scene_compare_colors(+Objs, -Colors)
    Sorted list of distinct colors in the object list.

    ?- scene_compare_colors([obj(r,[r(0,0)]), obj(b,[r(0,1)]), obj(r,[r(0,2)])], C).
    C = [b, r].

scene_compare_forms(+Objs, -Forms)
    Sorted list of distinct normalized shapes. Each form is a sorted cell list
    translated to the origin.

    ?- scene_compare_forms([obj(r,[r(0,0)]), obj(b,[r(0,1)])], F).
    F = [[r(0,0)]].   % both are single-cell dots at origin after normalization


Symmetric Comparison Predicates

scene_compare_same_n_objs(+Before, +After)
    Succeeds iff both lists have the same number of objects.

scene_compare_same_total_cells(+Before, +After)
    Succeeds iff both lists have the same total cell count.

scene_compare_same_colors(+Before, +After)
    Succeeds iff both lists have exactly the same sorted color set.

    ?- scene_compare_same_colors([obj(r,[r(0,0)]), obj(b,[r(0,1)])],
                       [obj(r,[r(1,0)]), obj(b,[r(1,1)])]).
    true.

scene_compare_same_forms(+Before, +After)
    Succeeds iff both lists have exactly the same sorted form set.


Asymmetric Comparison Predicates

scene_compare_added_colors(+Before, +After, -Added)
    Colors present in After but not in Before.

    ?- scene_compare_added_colors([obj(r,[r(0,0)])], [obj(r,[r(0,0)]), obj(b,[r(0,1)])], A).
    A = [b].

scene_compare_removed_colors(+Before, +After, -Removed)
    Colors present in Before but not in After.

scene_compare_added_forms(+Before, +After, -Added)
    Normalized forms present in After but not in Before.

scene_compare_removed_forms(+Before, +After, -Removed)
    Normalized forms present in Before but not in After.

scene_compare_n_color_change(+Before, +After, -N)
    N = |added colors| + |removed colors|. Zero means the same palette. One means
    exactly one color was added with none removed, or vice versa. Two means a color
    swap (one added, one removed).

    ?- scene_compare_n_color_change([obj(r,[r(0,0)]), obj(b,[r(0,1)])],
                          [obj(r,[r(0,0)]), obj(g,[r(1,0)])], N).
    N = 2.   % b removed, g added

scene_compare_any_change(+Before, +After)
    True iff the object count, color set, or form set differs between Before and
    After. Fails (does not raise an error) when no difference is detected.

    ?- scene_compare_any_change([obj(r,[r(0,0)])], [obj(r,[r(0,0)]), obj(b,[r(0,1)])]).
    true.   % count changed from 1 to 2


Worked Example: Detecting a Recoloring

Suppose the input scene has two dots - one red, one blue - and the output scene
has two dots - one red, one green. Has anything changed?

  Before = [obj(r,[r(0,0)]), obj(b,[r(0,1)])]
  After  = [obj(r,[r(0,0)]), obj(g,[r(1,0)])]

  ?- scene_compare_same_n_objs(Before, After).
  true.   % both have 2 objects

  ?- scene_compare_same_forms(Before, After).
  true.   % both have only the dot form

  ?- scene_compare_same_colors(Before, After).
  false.  % Before: {r,b}; After: {r,g}

  ?- scene_compare_added_colors(Before, After, Added).
  Added = [g].

  ?- scene_compare_removed_colors(Before, After, Removed).
  Removed = [b].

  ?- scene_compare_n_color_change(Before, After, N).
  N = 2.  % one color replaced by another

  ?- scene_compare_any_change(Before, After).
  true.

Summary

The scenecmp pack gives a reasoner a fast, high-level view of what changed between
two scenes. It does not find which specific objects changed, but it detects whether
anything changed, what colors appeared or disappeared, and what forms appeared or
disappeared. This makes it a useful first filter before more detailed analysis with
objmatch or objdelta.
