Chapter 92: Value Frequency Analysis - The Histogram Pack (hi_* prefix)

The histogram pack (Work Package 131, Layer 110) provides 14 predicates for
value frequency analysis of integer lists and 2D grids. It counts occurrences,
computes sorted frequency tables, finds the most and least common values, tests
for value presence and uniformity, selects the top-N most frequent values,
ranks values by frequency, and computes modal colors from grids.


1. Loading the Pack

:- use_module(packs/histogram/prolog/histogram).


2. Counting and Frequency Tables

histogram_count(+List, +V, -N) counts how many times V appears in List. Returns 0
when V is not present.

histogram_freq(+List, -Pairs) computes a V-Count pair list, sorted by value ascending.

?- histogram_count([1,2,1,3,1], 1, N).
N = 3.

?- histogram_freq([3,1,2,1,3,3], Pairs).
Pairs = [1-2, 2-1, 3-3].

?- histogram_freq([], Pairs).
Pairs = [].


3. Most and Least Common Values

histogram_mode(+List, -V, -N) finds the most frequent value and its count. Ties are
broken by smallest value.

histogram_rare(+List, -V, -N) finds the least frequent value. Ties are broken by
largest value.

?- histogram_mode([1,2,2,3,3,3], V, N).
V = 3, N = 3.

?- histogram_rare([1,2,2,3,3,3], V, N).
V = 1, N = 1.

?- histogram_mode([1,1,2,2], V, N).
V = 1, N = 2.   % tie: smallest wins


4. Distinct Values

histogram_unique_vals(+List, -Vals) returns the sorted list of distinct values.
histogram_unique_count(+List, -N) returns the count of distinct values.

?- histogram_unique_vals([3,1,2,1,3], Vals).
Vals = [1,2,3].

?- histogram_unique_count([1,1,2,3,3], N).
N = 3.


5. Sorting by Frequency

histogram_sorted_by_freq(+List, -Pairs) returns V-Count pairs sorted by count
descending. Ties in count are broken by value ascending.

?- histogram_sorted_by_freq([1,2,2,3,3,3], Pairs).
Pairs = [3-3, 2-2, 1-1].

?- histogram_sorted_by_freq([1,1,2,2], Pairs).
Pairs = [1-2, 2-2].   % tie: both count 2, smaller V first

The implementation negates counts and uses keysort (stable) to produce
descending-count order while preserving value-ascending order for ties.


6. Top-N Values

histogram_top_n(+List, +N, -TopN) returns the N most frequent V-Count pairs.

?- histogram_top_n([1,2,2,3,3,3], 2, TopN).
TopN = [3-3, 2-2].

?- histogram_top_n([1,2,3], 1, TopN).
TopN = [1-1].   % all equal frequency; smallest value first


7. Presence and Uniformity

histogram_has_value(+List, +V) succeeds if V appears at least once. Uses memberchk
for a single deterministic check.

histogram_all_equal(+List, -V) succeeds if every element equals V; binds V to the
first element. Fails for the empty list.

?- histogram_has_value([1,2,3], 2).
true.

?- histogram_all_equal([3,3,3], V).
V = 3.

?- histogram_all_equal([3,3,4], _).
false.


8. Grid-Level Predicates

histogram_grid_freq(+Grid, -Pairs) computes the frequency table of all cell values
in a 2D grid. Flattens the grid with append/2, then calls histogram_freq.

histogram_grid_mode(+Grid, -Color, -Count) finds the most frequent cell color.
histogram_grid_rare(+Grid, -Color, -Count) finds the least frequent cell color.

?- histogram_grid_freq([[1,2],[2,3]], Pairs).
Pairs = [1-1, 2-2, 3-1].

?- histogram_grid_mode([[1,2],[2,2]], Color, Count).
Color = 2, Count = 3.

?- histogram_grid_rare([[1,2,2,2]], Color, Count).
Color = 1, Count = 1.


9. Ranking

histogram_rank(+List, +V, -Rank) gives the 1-based rank of value V by frequency.
Rank 1 is the most frequent. Fails if V does not appear in List.

?- histogram_rank([1,2,2,3,3,3], 3, R).
R = 1.   % most common

?- histogram_rank([1,2,2,3,3,3], 1, R).
R = 3.   % least common


10. Summary

The histogram pack provides comprehensive frequency analysis for integer
lists and grids. Together with signal_row_profile and signal_col_profile from the
signal pack, it enables: extract a row as a 1D list, compute its frequency
table, find the dominant color, check if the row is uniform.

Predicate count: 14 (hi_* prefix). Tests: 42/42. Layer: 110.
