layout_stack solves ARC-AGI-2 task 291dc1e1. The input grid has a corner
cell of value 0 flanked by a row of 1s in one direction and 2s in the
perpendicular direction. These markers encode the reading order and
orientation of colored blocks embedded in the interior of the grid (after
stripping the header row and column). All blocks are extracted in reading
order, optionally rotated 90 degrees CCW (in column-major mode), and
stacked vertically in the output, centered within the width of the widest
block. The background color is 8.

The two reading modes:

Row-major (1s horizontal): The header row contains 1s. Row bands (groups
of consecutive rows with non-8 content) are scanned in the direction of the
2s (T->B or B->T). Within each row band, column stretches (consecutive
groups of cols with non-8 content) are scanned in the direction of the 1s
(L->R or R->L). Blocks are extracted as-is (no rotation).

Col-major (1s vertical): The header col contains 1s. Column bands (groups
of consecutive cols with non-8 content) are scanned in the direction of the
2s (L->R or R->L). Within each column band, row stretches (consecutive
groups of rows with non-8 content) are scanned T->B or B->T. Each block is
rotated 90 degrees CCW before stacking. This normalizes tall blocks (H x W)
into wide blocks (W x H).

The 90-degree CCW rotation formula: Output[i][j] = Input[j][C-1-i], where
C is the number of columns in the input block. This maps each input column
to an output row, reversing the column order.

After all blocks are collected, the output width is the maximum block width.
Each block row is centered by adding equal 8-padding on both sides, with any
odd remainder going to the right.

Algorithm:
  1. Find corner (row CR, col CC) where Grid[CR][CC] = 0.
  2. Read cell adjacent to corner along header row to get mode (1 = row,
     2 = col).
  3. PrimReverse = (CR == last row). SecReverse = (CC == last col).
  4. Strip header row and col to get Interior.
  5. Find bands (col bands for col-major, row bands for row-major).
  6. Apply SecReverse to band order if needed.
  7. For each band: find stretches; apply PrimReverse; extract sub-grids.
  8. For col-major: rotate each sub-grid 90 CCW.
  9. Compute MaxW = max block width.
  10. Center each block row in MaxW with 8-padding.
  11. Concatenate all centered rows into the output grid.
