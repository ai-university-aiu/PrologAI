noise_erase solves ARC-AGI-2 task 71e489b6. The input is a binary grid
containing only 0s and 1s. Two large regions exist: a 0-region and a
1-region. Scattered through both regions are noise cells that must be
identified and treated differently depending on which region they invade.


Algorithm:

  Phase 1 - Detect stray-0 seeds:
  1. For each 0-cell (R,C), count how many of its in-bounds 4-neighbors
     hold value 1 (call this OneCount) and how many 4-neighbors exist in
     bounds at all (call this Avail).
  2. If OneCount >= min(3, Avail), mark (R,C) as a stray-0 seed.
     The min(3, Avail) threshold adapts to corner cells (Avail=2) and
     edge cells (Avail=3), classifying them as stray when all or nearly
     all available neighbors are 1.

  Phase 2 - Expand the stray-0 set by BFS:
  3. Start a BFS queue with all seeds.
  4. For each position in the queue, examine its 4-connected 0-cell
     neighbors that are not yet in the stray-0 set.
  5. If such a neighbor itself has >= 2 1-valued 4-neighbors, add it to
     the stray-0 set and to the BFS queue.
  6. Repeat until the queue is empty.

  Phase 3 - Build the frame:
  7. For every stray-0 cell, collect all in-bounds 8-neighbors.
  8. Remove positions that are themselves stray-0 cells.
  9. The remaining positions form the frame; they will become 7.

  Phase 4 - Detect stray-1 cells:
  10. For each 1-cell (R,C), count 1-valued 4-neighbors.
  11. If that count is <= 1, mark (R,C) as stray-1.

  Phase 5 - Build output:
  12. If (R,C) is a stray-0 center: output 0 (cell is preserved).
  13. Else if (R,C) is a frame cell: output 7.
  14. Else if (R,C) is a stray-1: output 0 (cell is erased).
  15. Else: copy the input value.

Key insight:

A single per-cell threshold fails for stray-0 clusters: interior cluster
cells have only 2 1-neighbors (the other neighbors are fellow cluster cells).
The two-phase approach solves this by first seeding with cells that clearly
qualify (>=3 or all-of-fewer 1-neighbors) and then BFS-propagating to
reachable cluster cells. BFS does not contaminate the main 0-region because
its boundary cells have at most 1 1-neighbor.
