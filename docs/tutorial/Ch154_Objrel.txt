154.1 Overview

The objrel pack answers questions about the relationship between two
obj(Color, Cells) terms: do they share cells? are they adjacent? does
one contain the other? how far apart are their centers? which direction
is one from the other?

Colors are ignored in every predicate. All reasoning is based on cell
positions (r(Row, Col) terms) only.

The pack has 14 exported predicates (or_* prefix) and three private
helpers: or_centroid_, or_bbox_, and or_dir_.

Layer: 172. No dependencies on other packs.

154.2 Overlap and Shared Cells

or_overlap(+Obj1, +Obj2)

Succeeds if Obj1 and Obj2 share at least one cell position.

    A = obj(a, [r(0,0), r(0,1)]),
    B = obj(b, [r(0,1), r(0,2)]),
    or_overlap(A, B).        % true: both contain r(0,1)

or_shared_cells(+Obj1, +Obj2, -Shared)

Shared is the list of cells present in both objects.

    or_shared_cells(A, B, Shared).
    % Shared = [r(0,1)]

or_n_shared(+Obj1, +Obj2, -N)

N is the count of shared cells.

    or_n_shared(A, B, N).    % N = 1

154.3 Adjacency

or_touch4(+Obj1, +Obj2)

Succeeds if the objects are 4-adjacent (Manhattan distance 1 between
some pair of cells) but do NOT overlap.

    Left  = obj(l, [r(0,0)]),
    Right = obj(r, [r(0,1)]),
    or_touch4(Left, Right).      % true: cells are east-west neighbors

or_touch8(+Obj1, +Obj2)

Succeeds if the objects are 8-adjacent (Chebyshev distance 1, which
includes diagonals) but do NOT overlap.

    A  = obj(a, [r(0,0)]),
    SE = obj(s, [r(1,1)]),
    or_touch8(A, SE).            % true: diagonal neighbor

154.4 Containment and Union

or_contains(+Obj1, +Obj2)

Succeeds if every cell of Obj2 is also a cell of Obj1. Obj1 contains
Obj2 as a subset.

    Big  = obj(g, [r(0,0),r(0,1),r(1,0),r(1,1)]),
    Tiny = obj(t, [r(0,0)]),
    or_contains(Big, Tiny).      % true

or_union_cells(+Obj1, +Obj2, -Union)

Union is the sorted list of all cell positions appearing in either
object. Duplicates from overlap are removed.

    H = obj(h, [r(0,0),r(0,1),r(0,2)]),
    V = obj(v, [r(0,0),r(1,0),r(2,0)]),
    or_union_cells(H, V, Union).
    % Union = [r(0,0),r(0,1),r(0,2),r(1,0),r(2,0)]  (5 cells)

154.5 Distance

or_dist(+Obj1, +Obj2, -D)

D is the Euclidean distance between centroids. Centroids are
(sum_rows/N, sum_cols/N) and D is a float.

    A = obj(a, [r(0,0)]),
    B = obj(b, [r(3,4)]),
    or_dist(A, B, D).        % D = 5.0

or_manhattan(+Obj1, +Obj2, -D)

D is the Manhattan distance between centroids.

    or_manhattan(A, B, D).   % D = 7.0  (|3-0| + |4-0|)

154.6 Direction

or_direction(+Obj1, +Obj2, -Dir)

Dir is the compass direction from Obj1's centroid to Obj2's centroid.
Dir is one of: n, s, e, w, ne, nw, se, sw, or same.

Rows increase southward; columns increase eastward.

    North = obj(n, [r(0,2)]),
    South = obj(s, [r(4,2)]),
    or_direction(South, North, Dir).   % Dir = n
    or_direction(North, South, Dir2).  % Dir2 = s

154.7 Alignment

or_aligned_h(+Obj1, +Obj2)

Succeeds if both objects have the same centroid row.

    Row2a = obj(a, [r(2,0)]),
    Row2b = obj(b, [r(2,5)]),
    or_aligned_h(Row2a, Row2b).   % true

or_aligned_v(+Obj1, +Obj2)

Succeeds if both objects have the same centroid column.

    Col3a = obj(a, [r(0,3)]),
    Col3b = obj(b, [r(7,3)]),
    or_aligned_v(Col3a, Col3b).   % true

154.8 Bounding Box Gaps

or_gap_rows(+Obj1, +Obj2, -G)

G is the number of empty rows between the bounding boxes of Obj1 and
Obj2. G = 0 when bounding boxes touch or overlap in the row dimension.

    Top = obj(t, [r(0,0)]),
    Bot = obj(b, [r(3,0)]),
    or_gap_rows(Top, Bot, G).    % G = 2  (rows 1 and 2 are empty between)

or_gap_cols(+Obj1, +Obj2, -G)

G is the number of empty columns between the bounding boxes.

    Left  = obj(l, [r(0,0)]),
    Right = obj(r, [r(0,4)]),
    or_gap_cols(Left, Right, G). % G = 3  (cols 1, 2, 3 are empty between)
