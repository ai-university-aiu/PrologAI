1. Overview

The gridframe pack (Layer 216) treats a grid as a set of concentric rings,
called frames. Frame depth D is the set of all cells (R,C) whose minimum
distance to any grid border equals D. Frame 0 is the outermost border;
frame 1 is the second ring in; and so on.

Load the pack:

    :- use_module(pack(gridframe/prolog/gridframe)).


2. Grid Format

Raw grids are lists of rows, each a list of color atoms, 0-indexed.


3. Frame Depth

The depth of a cell (R,C) in an H x W grid is:

    D = min(R, H-1-R, C, W-1-C)

This equals the cell's minimum distance to any of the four borders.

For a 5x5 grid:
  - Corner (0,0): min(0, 4, 0, 4) = 0  (on the border)
  - Edge (0,2):   min(0, 4, 2, 2) = 0  (still on the border)
  - Ring1 (1,1):  min(1, 3, 1, 3) = 1
  - Center (2,2): min(2, 2, 2, 2) = 2

The maximum frame depth for an H x W grid is floor((min(H,W)-1)/2).


4. Cell Depth

gfr_cell_depth/4:

    G = [[a,b,c],[d,e,f],[g,h,i]].
    gfr_cell_depth(G, 0, 0, D).
    % D = 0   (corner)

    gfr_cell_depth(G, 1, 1, D).
    % D = 1   (center of 3x3)

    G5 = [[...],...].   % 5x5
    gfr_cell_depth(G5, 2, 2, D).
    % D = 2   (center of 5x5)


5. Frame Extraction

gfr_frame/3 returns cells at a given depth as R-C pairs:

    G = [[a,b,c],[d,e,f],[g,h,i]].   % 3x3
    gfr_frame(G, 0, Cells).
    % Cells = [0-0, 0-1, 0-2, 1-0, 1-2, 2-0, 2-1, 2-2]   (8 border cells)

    gfr_frame(G, 1, Cells).
    % Cells = [1-1]   (center)

gfr_max_depth/2 and gfr_frame_count/2:

    gfr_max_depth(G, D).
    % D = 1   (for 3x3)

    gfr_frame_count(G, N).
    % N = 2   (depths 0 and 1)

    G5 = [[...],...]   % 5x5
    gfr_max_depth(G5, D).
    % D = 2

    gfr_frame_count(G5, N).
    % N = 3

gfr_all_frames/2 returns all frame cell-lists:

    gfr_all_frames(G, Frames).
    % Frames = [[0-0,...,2-2], [1-1]]  (frame 0 and frame 1)


6. Peeling

gfr_peel/2 removes the outermost frame, returning the (H-2)x(W-2) inner grid:

    Ring = [[r,r,r,r,r],
            [r,x,x,x,r],
            [r,x,b,x,r],
            [r,x,x,x,r],
            [r,r,r,r,r]].
    gfr_peel(Ring, Inner).
    % Inner = [[x,x,x],[x,b,x],[x,x,x]]   (3x3 inner)

    gfr_peel(Inner, Core).
    % Core = [[b]]   (the center cell)

Note: gfr_peel fails if the grid has fewer than 3 rows or 3 columns.


7. Frame Uniformity and Colors

gfr_frame_uniform/3 tests whether a frame is all one color:

    Ring = [[r,r,r],[r,x,r],[r,r,r]].
    gfr_frame_uniform(Ring, 0, Bool).
    % Bool = yes   (all r)

    gfr_frame_uniform(Ring, 1, Bool).
    % Bool = yes   (center is x)

    G = [[a,b,c],[d,e,f],[g,h,i]].
    gfr_frame_uniform(G, 0, Bool).
    % Bool = no   (border has many distinct values)

gfr_frame_colors/3 returns distinct colors in a frame:

    gfr_frame_colors(Ring, 0, Colors).
    % Colors = [r]

    gfr_frame_colors(G, 0, Colors).
    % Colors = [a, b, c, d, f, g, h, i]   (all distinct border values)


8. Counting and Uniformity Collection

gfr_count_in_frame/4 counts a specific color in a frame:

    Ring = [[r,r,r],[r,x,r],[r,r,r]].
    gfr_count_in_frame(Ring, 0, r, Count).
    % Count = 8

gfr_uniform_frames/2 collects all depths with uniform frames:

    Ring5 = [[r,r,r,r,r],[r,x,x,x,r],[r,x,b,x,r],[r,x,x,x,r],[r,r,r,r,r]].
    gfr_uniform_frames(Ring5, Ds).
    % Ds = [0, 1, 2]


9. Modification

gfr_set_frame/4 sets all cells at depth D to a given Color:

    G = [[r,r,r],[r,x,r],[r,r,r]].
    gfr_set_frame(G, 0, y, Result).
    % Result = [[y,y,y],[y,x,y],[y,y,y]]

gfr_fill_frames/3 fills each frame with a color from a list:

    G = [[a,b,c],[d,e,f],[g,h,i]].
    gfr_fill_frames(G, [r, x], Result).
    % Result = [[r,r,r],[r,x,r],[r,r,r]]

    G5 = [[...,...],...]   % 5x5 grid
    gfr_fill_frames(G5, [r, x, b], Result).
    % Result = [[r,r,r,r,r],[r,x,x,x,r],[r,x,b,x,r],[r,x,x,x,r],[r,r,r,r,r]]


10. Innermost Frame and Onion Patterns

gfr_innermost/2 returns cells at the maximum depth:

    G = [[a,b,c],[d,e,f],[g,h,i]].
    gfr_innermost(G, Cells).
    % Cells = [1-1]   (the center)

    G5 = [[...],...].   % 5x5
    gfr_innermost(G5, Cells).
    % Cells = [2-2]   (center)

gfr_onion_layers/2 returns D-Color pairs for each uniform frame:

    Ring = [[r,r,r],[r,x,r],[r,r,r]].
    gfr_onion_layers(Ring, Layers).
    % Layers = [0-r, 1-x]

    Ring5 = [[r,r,r,r,r],[r,x,x,x,r],[r,x,b,x,r],[r,x,x,x,r],[r,r,r,r,r]].
    gfr_onion_layers(Ring5, Layers).
    % Layers = [0-r, 1-x, 2-b]

A grid is a "perfect onion" when the length of gfr_onion_layers equals gfr_frame_count.

12. Summary

gfr_cell_depth/4      - frame depth of a specific cell
gfr_frame/3           - cells at frame depth D
gfr_max_depth/2       - maximum frame depth: floor((min(H,W)-1)/2)
gfr_frame_count/2     - number of distinct frame depths
gfr_peel/2            - inner (H-2)x(W-2) grid after removing outermost frame
gfr_all_frames/2      - all frame cell-lists from depth 0 to max_depth
gfr_frame_uniform/3   - yes/no uniformity of frame D
gfr_frame_colors/3    - distinct colors in frame D
gfr_set_frame/4       - set frame D to Color
gfr_count_in_frame/4  - count Color at frame depth D
gfr_uniform_frames/2  - list of depths with uniform frames
gfr_fill_frames/3     - fill each frame with Colors[D]
gfr_innermost/2       - cells at maximum depth
gfr_onion_layers/2    - D-Color pairs for uniform frames (onion pattern)
