Chapter 217: gridblend - Grid Blending and Layered Composition (gbld_*, Layer 235)

Overview

The gridblend pack provides ten gbld_* predicates for compositing grids in layers,
applying spatial blend patterns, and reducing lists of grids to a single composite.
All operations treat one designated BgColor as transparent: a bg cell is "absent"
and the corresponding cell from another layer shows through.

Pack: gridblend
Module: gridblend
Prefix: gbld_
Layer: 235
Dependencies: library(lists), msort/2 (built-in)


Overlay and Underlay

gridblend_overlay(+Top, +Bottom, +BgColor, -Result)
Non-bg cells in Top always show; bg cells in Top reveal Bottom.

    Top    = [[r,b],[b,b]], Bottom = [[g,g],[g,g]].
    gridblend_overlay(Top, Bottom, b, [[r,g],[g,g]]).

gridblend_underlay(+Top, +Bottom, +BgColor, -Result)
Non-bg cells in Bottom always show; bg cells in Bottom reveal Top.

    Top = [[r,r],[r,r]], Bottom = [[g,b],[b,b]].
    gridblend_underlay(Top, Bottom, b, [[g,r],[r,r]]).


Stencil Masking

gridblend_stencil(+Source, +Stencil, +StencilColor, -Result)
Keep Source's cell only where Stencil = StencilColor. Other positions output
StencilColor (acts as the "erased" background).

    Source  = [[r,g],[g,r]], Stencil = [[m,b],[b,m]].
    gridblend_stencil(Source, Stencil, m, [[r,m],[m,r]]).


Priority Reduction and Merge

gridblend_priority(+Grids, +BgColor, -Result)
Reduce a list of grids with the first as topmost: first non-bg value at each
position wins across the list.

    G1 = [[r,b],[b,b]], G2 = [[g,g],[b,b]], G3 = [[y,y],[y,y]].
    gridblend_priority([G1,G2,G3], b, [[r,g],[y,y]]).

gridblend_merge_many(+Grids, +BgColor, -Result)
Reduce a list of grids with the last as topmost (first grid is the base layer).
Later grids paint over earlier ones.

    G1 = [[r,b],[b,b]], G2 = [[b,g],[b,b]], G3 = [[b,b],[y,b]].
    gridblend_merge_many([G1,G2,G3], b, [[r,g],[y,b]]).


Spatial Blend Patterns

gridblend_checker_blend(+Grid1, +Grid2, +BgColor, +Parity, -Result)
Blend using checkerboard parity. Positions where (R+C) mod 2 = Parity take from
Grid1; all others from Grid2. Parity is 0 or 1.

    gridblend_checker_blend([[r,r],[r,r]], [[g,g],[g,g]], b, 0, [[r,g],[g,r]]).

gridblend_stripe_blend(+Grid1, +Grid2, +BgColor, +StripeWidth, -Result)
Blend using horizontal row stripes of StripeWidth rows each. Even stripe bands
use Grid1; odd bands use Grid2. Within each band, one grid overlays the other.

    gridblend_stripe_blend([[r,r],[r,r],[r,r],[r,r]], [[g,g],[g,g],[g,g],[g,g]], b, 2,
        [[r,r],[r,r],[g,g],[g,g]]).


Threshold and Voting

gridblend_threshold_replace(+Grid, +Threshold, +ReplaceColor, +BgColor, -Result)
In each row: if the count of non-bg cells is >= Threshold, replace every non-bg
cell with ReplaceColor. Rows below the threshold are unchanged.

    gridblend_threshold_replace([[r,g,r],[b,b,b]], 3, x, b, [[x,x,x],[b,b,b]]).

gridblend_dominant(+Grids, +BgColor, -Result)
At each cell position, take the most frequent non-bg value across the grid list.
All-bg positions remain bg.

    G1 = [[r,b]], G2 = [[r,g]], G3 = [[b,g]].
    gridblend_dominant([G1,G2,G3], b, [[r,g]]).


Mode-Dispatched Composite

gridblend_composite(+Grids, +BgColor, +Mode, -Result)
Reduce a list of grids to one according to Mode:
  overlay   - first grid on top (gridblend_priority)
  underlay  - last grid on top  (gridblend_merge_many)
  dominant  - most frequent non-bg per cell (gridblend_dominant)

    gridblend_composite([G1,G2,G3], b, overlay,  Result).
    gridblend_composite([G1,G2,G3], b, underlay, Result).
    gridblend_composite([G1,G2,G3], b, dominant, Result).


Typical Workflows

% Layer two grids with transparency:
gridblend_overlay(ForegroundGrid, BackgroundGrid, Bg, Composite).

% Protect a region using a stencil mask:
gridblend_stencil(Source, MaskGrid, MaskColor, Protected).

% Build a composite from multiple candidates, first answer wins:
gridblend_priority(Candidates, Bg, BestGuess).

% Find consensus across multiple transformation outputs:
gridblend_dominant(Outputs, Bg, Consensus).
