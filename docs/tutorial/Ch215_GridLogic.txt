Chapter 215: gridlogic - Grid Logical Operations (ggl_*, Layer 233)

Overview

The gridlogic pack provides fourteen ggl_* predicates for cell-wise logical and
set-theoretic operations on symbolic grids. One designated BgColor acts as the
logical "false" value; all other colors are "true". Predicates cover pairwise
binary operations (AND, OR, XOR, NOT, subtract, common, differ), list-wide
reductions (any, all, majority, unanimous), and conditional/filtering operations
(mask, if-then-else, filter).

Pack: gridlogic
Module: gridlogic
Prefix: ggl_
Layer: 233
Dependencies: library(lists)


Pairwise Binary Operations

gridlogic_and(+Grid1, +Grid2, +BgColor, -Result)
Both cells non-bg → Grid1's value. Otherwise bg.

    A = [[r,b],[b,b]], B = [[r,b],[r,b]].
    gridlogic_and(A, B, b, R).
    % R = [[r,b],[b,b]]   (only (0,0) is non-bg in both)

gridlogic_or(+Grid1, +Grid2, +BgColor, -Result)
First non-bg wins (Grid1 over Grid2). If both bg → bg.

    gridlogic_or([[r,b],[b,b]], [[b,g],[b,b]], b, R).
    % R = [[r,g],[b,b]]

gridlogic_xor(+Grid1, +Grid2, +BgColor, -Result)
Exactly one non-bg → that value. Both bg or both non-bg → bg.

    gridlogic_xor([[r,b],[b,b]], [[r,g],[b,b]], b, R).
    % R = [[b,g],[b,b]]   ((0,0) is non-bg in both → bg)

gridlogic_not(+Grid, +BgColor, +FgColor, -Result)
Logical NOT: bg → FgColor; non-bg → bg.

    gridlogic_not([[r,b],[b,g]], b, x, R).
    % R = [[b,x],[x,b]]

gridlogic_subtract(+Grid1, +Grid2, +BgColor, -Result)
Grid1 cells that are absent from Grid2: Grid1 non-bg AND Grid2 is bg → Grid1 value.

    gridlogic_subtract([[r,b],[b,g]], [[r,b],[b,b]], b, R).
    % R = [[b,b],[b,g]]   (r is in both; only g is unique to Grid1)

gridlogic_common(+Grid1, +Grid2, +BgColor, -Result)
Cells where both grids agree on the same non-bg value → that value. Else bg.

    gridlogic_common([[r,b],[b,g]], [[r,b],[r,b]], b, R).
    % R = [[r,b],[b,b]]

gridlogic_differ(+Grid1, +Grid2, +BgColor, -Result)
Cells where both grids are non-bg but have different values → Grid1's value. Else bg.

    gridlogic_differ([[r,b],[b,g]], [[x,b],[b,g]], b, R).
    % R = [[r,b],[b,b]]   ((0,0) r≠x; (1,1) g=g so not a difference)


List-Wide Reductions

gridlogic_any(+Grids, +BgColor, -Result)
First non-bg value at each position across the list wins. Equivalent to fold-or.

    gridlogic_any([[[r,b],[b,b]], [[b,b],[b,g]]], b, R).
    % R = [[r,b],[b,g]]

gridlogic_all(+Grids, +BgColor, -Result)
Non-bg value if all grids in the list agree at that position. Equivalent to fold-and.

    gridlogic_all([[[r,b],[b,g]], [[r,b],[b,g]]], b, R).
    % R = [[r,b],[b,g]]
    gridlogic_all([[[r,b],[b,g]], [[r,b],[b,x]]], b, R).
    % R = [[r,b],[b,b]]   (disagreement at (1,1))

gridlogic_majority(+Grids, +BgColor, -Result)
Plurality vote: most frequent non-bg at each cell if count > N/2; else bg.

    gridlogic_majority([G, G, H], b, R).  % 2 of 3 agree at r-position → r wins

gridlogic_unanimous(+Grids, +BgColor, -Result)
Strict all-grids agreement including bg cells. Any disagreement → bg.


Conditional and Filtering

gridlogic_mask(+Grid, +MaskGrid, +MaskColor, +BgColor, -Result)
Keep Grid's cell where MaskGrid = MaskColor; all others → BgColor.

    gridlogic_mask([[r,g],[g,r]], [[m,b],[b,m]], m, b, R).
    % R = [[r,b],[b,r]]

gridlogic_if(+CondGrid, +CondColor, +ThenGrid, +ElseGrid, -Result)
Cell-wise conditional: CondGrid = CondColor → ThenGrid cell; else ElseGrid cell.

    gridlogic_if([[c,b],[b,c]], c, [[r,g],[g,r]], [[x,y],[y,x]], R).
    % R = [[r,y],[y,r]]

gridlogic_filter(+Grid, +Colors, +BgColor, -Result)
Keep cells whose color is in Colors; replace all others with BgColor.

    gridlogic_filter([[r,g],[g,r]], [r], b, R).
    % R = [[r,b],[b,r]]


Typical Workflows

% Combine two layers with Grid2 on top of Grid1:
gridlogic_or(Grid2, Grid1, Bg, Composite).

% Find cells unique to each grid:
gridlogic_xor(Grid1, Grid2, Bg, Unique).

% Keep only cells present in all sample grids:
gridlogic_all(SampleGrids, Bg, Common).

% Majority vote from multiple transformation candidates:
gridlogic_majority(Candidates, Bg, Result).
