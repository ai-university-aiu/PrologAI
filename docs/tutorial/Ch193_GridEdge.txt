Chapter 193: gridedge Pack - Edge Detection, Boundary Analysis, Corners, Endpoints, and Transition Maps (ge_*, Layer 211)

Overview

The gridedge pack (Layer 211, ge_* prefix) provides fourteen predicates for
detecting edges and structural features of symbolic grids. An edge cell is one
whose color differs from at least one in-bounds 4-connected (or 8-connected)
neighbor. The pack covers 4- and 8-connected edge detection, directional
boundary detection, inner and outer border extraction, edge grid construction,
corner and endpoint detection, smooth cell detection, and transition maps.

Load the pack:

:- use_module(pack(gridedge/prolog/gridedge)).


1. Edge Cell Detection

gridedge_edge_cells(+Grid, +FgColor, -Cells)

Cells is the list of FgColor positions that have at least one 4-connected
neighbor of a different color. Each cell appears exactly once.

Example:
G = [[r,r,x],[r,r,x],[x,x,x]].
gridedge_edge_cells(G, r, Cells).
% Cells = [0-1, 1-0, 1-1]  (r cells touching x; corner 0-0 has only r neighbors)

gridedge_edge_cells8(+Grid, +FgColor, -Cells)

Like gridedge_edge_cells but uses the 8-connected (Moore) neighborhood.

gridedge_is_edge(+Grid, +R-C, -Bool)

Bool is 'yes' if cell (R,C) has at least one 4-connected neighbor of a
different color; 'no' otherwise.

Example:
gridedge_is_edge([[r,x],[x,r]], 0-0, B).
% B = yes


2. Boundary Detection

gridedge_boundary(+Grid, +Color1, +Color2, -Cells)

Cells is the list of Color1 positions that have at least one 4-connected
Color2 neighbor.

Example:
G = [[r,r,x],[r,r,x],[x,x,x]].
gridedge_boundary(G, r, x, Cells).
% Cells = [0-1, 1-0, 1-1]

gridedge_inner_border(+Grid, +BgColor, -Cells)

Cells is the list of BgColor positions adjacent to at least one non-BgColor
cell (the inner rim of the background).

gridedge_outer_border(+Grid, +FgColor, -Cells)

Cells is the list of non-FgColor positions adjacent to at least one FgColor
cell (the cells immediately outside the FgColor region).


3. Edge Grid and Counts

gridedge_edge_grid(+Grid, +FgColor, +EdgeColor, -Result)

Result is Grid with FgColor edge cells replaced by EdgeColor; all other cells
retain their original color.

Example:
gridedge_edge_grid([[r,r,x],[r,r,x],[x,x,x]], r, e, Result).
% Result = [[r,e,x],[e,e,x],[x,x,x]]

gridedge_edge_count(+Grid, +FgColor, -N)

N is the number of FgColor edge cells.

gridedge_edge_color_count(+Grid, +FgColor, +BgColor, -N)

N is the number of FgColor cells adjacent to at least one BgColor cell.


4. Neighbor Analysis

gridedge_neighbors_diff(+Grid, +R, +C, -Neighbors)

Neighbors is the list of R-C positions among the 4-connected neighbors of
(R,C) that have a different color from cell (R,C).

gridedge_corners(+Grid, +FgColor, -Cells)

Cells is the list of FgColor positions with 2 or more differently-colored
4-connected neighbors (corners of regions).

gridedge_endpoints(+Grid, +FgColor, -Cells)

Cells is the list of FgColor positions with exactly 1 same-color 4-connected
neighbor (line endpoints).

gridedge_smooth_cells(+Grid, +FgColor, -Cells)

Cells is the list of FgColor positions with no differently-colored in-bounds
4-connected neighbor (fully interior cells).


5. Transition Map

gridedge_transition_map(+Grid, -Result)

Result is a grid where each cell value is the count (0..4) of 4-connected
in-bounds neighbors with a different color. Interior uniform cells have value 0;
edge cells have 1, 2, 3, or 4.

Example:
G = [[r,r,x],[r,r,x],[x,x,x]].
gridedge_transition_map(G, R).
% R = [[0,1,1],[1,2,1],[1,1,0]]


6. Summary of Predicates

gridedge_edge_cells/3      - 4-connected edge cells of FgColor
gridedge_edge_cells8/3     - 8-connected edge cells of FgColor
gridedge_is_edge/3         - test if single cell is an edge cell
gridedge_boundary/4        - Color1 cells adjacent to Color2 cells
gridedge_inner_border/3    - BgColor cells adjacent to non-BgColor
gridedge_outer_border/3    - non-FgColor cells adjacent to FgColor
gridedge_edge_grid/4       - grid with edge cells replaced by EdgeColor
gridedge_edge_count/3      - count of FgColor edge cells
gridedge_neighbors_diff/4  - 4-connected neighbors with different color
gridedge_corners/3         - cells with 2+ differently-colored 4-neighbors
gridedge_endpoints/3       - cells with exactly 1 same-color 4-neighbor
gridedge_smooth_cells/3    - cells with no differently-colored 4-neighbors
gridedge_transition_map/2  - grid of per-cell differently-colored neighbor counts
gridedge_edge_color_count/4 - count FgColor cells adjacent to BgColor
