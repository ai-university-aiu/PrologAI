Chapter 169: condxf Pack - Conditional and Selective Scene Transformation (xc_*, Layer 187)


Overview

The condxf pack applies transformations to a subset of objects in an
obj(Color, Cells) scene list - those satisfying a condition - while leaving
the remaining objects unchanged. It complements the scenexf pack (sx_*), which
applies transformations uniformly to every object.

Layer 187. Pack prefix: xc_*. Work Package 208.


What "Conditional" Means

In scenexf, a shift or recolor affects every object in the scene. In condxf,
you specify which objects are affected by a condition: color identity, size
comparison, or position in a size ranking. Objects that do not meet the condition
are passed through unchanged.

This allows targeted operations such as:
- recolor only the red objects
- shift only objects with more than 2 cells
- recolor the single largest object


Exported Predicates


conditional_transform_split_color(+Scene, +Color, -Match, -NoMatch)

Partition Scene into two lists: Match contains all objects whose color is Color;
NoMatch contains all others. Order within each list is preserved.

Example:

% R = obj(r,[r(0,0)]), B = obj(b,[r(0,1)]), RB = obj(r,[r(0,0),r(0,1),r(0,2)])
conditional_transform_split_color([R, B, RB], r, Match, NoMatch).
% Match   = [R, RB]
% NoMatch = [B]


conditional_transform_merge(+Group1, +Group2, -Scene2)

Concatenate two object groups into one scene list (simple append). Useful
after conditional_transform_split_color to recombine the groups after modifying one of them.

Example:

conditional_transform_merge([obj(r,[r(0,0)])], [obj(b,[r(0,1)])], Scene2).
% Scene2 = [obj(r,[r(0,0)]), obj(b,[r(0,1)])]


conditional_transform_recolor_matching(+Scene, +MatchColor, +NewColor, -Scene2)

Recolor all objects whose color is MatchColor to NewColor. Other objects
are unchanged.

Example:

conditional_transform_recolor_matching([R, B, RB], r, g, Out).
% Out = [obj(g,[r(0,0)]), B, obj(g,[r(0,0),r(0,1),r(0,2)])]


conditional_transform_shift_color(+Scene, +Color, +DR, +DC, -Scene2)

Translate all objects of Color by (DR, DC). Other objects are unchanged.

Example:

conditional_transform_shift_color([R, B], r, 2, 3, Out).
% Out = [obj(r,[r(2,3)]), B]


conditional_transform_swap_colors(+Scene, +C1, +C2, -Scene2)

Swap colors C1 and C2 throughout the scene. Objects of C1 become C2, objects
of C2 become C1, and objects of any other color are unchanged.

Example:

conditional_transform_swap_colors([obj(r,[r(0,0)]), obj(b,[r(0,1)])], r, b, Out).
% Out = [obj(b,[r(0,0)]), obj(r,[r(0,1)])]


conditional_transform_recolor_by_size(+Scene, +N, +Cmp, +NewColor, -Scene2)

Recolor each object whose cell count satisfies the comparison Cmp against
threshold N to NewColor. Cmp is one of: gt (>), lt (<), eq (=:=), ge (>=),
le (=<). Objects not satisfying the comparison are unchanged.

Example:

% RB has 3 cells, R has 1 cell
conditional_transform_recolor_by_size([R, RB], 1, gt, y, Out).
% Only RB satisfies size > 1
% Out = [R, obj(y,[r(0,0),r(0,1),r(0,2)])]


conditional_transform_move_color(+Scene, +Color, +DR, +DC, -Scene2)

Alias for conditional_transform_shift_color. Provided for readability in contexts where "move"
is more natural than "shift".


conditional_transform_largest(+Scene, -Obj)

The single largest object by cell count. On ties, the object that sorts first
under standard Prolog term order is returned, giving deterministic behavior.

Example:

conditional_transform_largest([obj(r,[r(0,0)]), obj(r,[r(0,0),r(0,1),r(0,2)])], Obj).
% Obj = obj(r,[r(0,0),r(0,1),r(0,2)])  (3 cells > 1 cell)


conditional_transform_smallest(+Scene, -Obj)

The single smallest object by cell count. On ties, standard term order
determines which object is returned.


conditional_transform_recolor_largest(+Scene, +NewColor, -Scene2)

Recolor the largest object (as found by conditional_transform_largest) to NewColor. Other
objects are unchanged.

Example:

conditional_transform_recolor_largest([R, RB], g, Out).
% Out = [R, obj(g,[r(0,0),r(0,1),r(0,2)])]


conditional_transform_recolor_smallest(+Scene, +NewColor, -Scene2)

Recolor the smallest object (as found by conditional_transform_smallest) to NewColor. Other
objects are unchanged.


conditional_transform_unique_size(+Scene, -Obj)

Backtrackable. Succeeds for each object whose cell count appears exactly once
in the scene. If every size appears more than once, the predicate fails.

Example:

% R and B both have 1 cell; L has 3 cells (unique)
conditional_transform_unique_size([R, B, L], L).


conditional_transform_recolor_unique_size(+Scene, +NewColor, -Scene2)

Recolor the unique-size object to NewColor. Fails if no unique-size object
exists (because conditional_transform_unique_size fails).

Example:

conditional_transform_recolor_unique_size([R, B, L], g, Out).
% Out = [R, B, obj(g,[r(0,0),r(1,0),r(1,1)])]


conditional_transform_split_size(+Scene, +N, +Cmp, -Match, -NoMatch)

Partition Scene by a size comparison (same Cmp atoms as conditional_transform_recolor_by_size).
Match contains objects satisfying size Cmp N; NoMatch contains the rest.

Example:

conditional_transform_split_size([R, RB], 1, gt, Match, NoMatch).
% Match   = [RB]  (3 > 1)
% NoMatch = [R]   (1 is not > 1)


Typical Workflow

A common use is to isolate one group, transform it, and recombine:

% Isolate the red objects, shift them down by 1 row, recombine.
conditional_transform_split_color(Scene, r, Reds, Others),
conditional_transform_shift_color(Reds, r, 1, 0, ShiftedReds),
conditional_transform_merge(ShiftedReds, Others, Scene2).

Or, more directly:

conditional_transform_shift_color(Scene, r, 1, 0, Scene2).


Private Helpers

conditional_transform_color_/2       - extract color from obj
conditional_transform_size_/2        - cell count of obj
conditional_transform_size_cmp_/3    - compare size with threshold using Cmp atom
conditional_transform_recolor_if_match_/4  - recolor one obj if color matches
conditional_transform_shift_if_color_/5    - shift one obj if color matches
conditional_transform_swap_one_/4    - swap C1<->C2 in one obj using if-then-else
conditional_transform_recolor_if_size_/5   - recolor one obj if size satisfies Cmp
conditional_transform_recolor_if_obj_/4    - recolor one obj if it is the target identity

Summary

The condxf pack provides selective scene transformation: apply any color or
spatial operation to a subset of objects chosen by color, size, size rank, or
uniqueness of size. Combined with scenexf (uniform transforms), ruleinfer (rule
inference), and sceneapply (rule dispatch), it completes the toolkit for
symbolic scene reasoning.
