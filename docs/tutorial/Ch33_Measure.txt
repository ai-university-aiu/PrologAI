Chapter 33: Measure Pack

The measure pack (module measure, prefix ms_) provides geometric measurements
of cell regions and grids. A region is any list of r(R,C) cells. All results
are deterministic integers. Layer 51, requires grid.

Load the pack:

    :- use_module(library(measure)).


Area

measure_area/2 returns the number of cells in a region.

    ?- measure_area([r(0,0), r(0,1), r(1,0), r(1,1)], N).
    N = 4.


Bounding Box

measure_bbox/2 returns the tight bounding box of a region as bbox(R1,C1,R2,C2)
with inclusive row/col bounds.

    ?- measure_bbox([r(1,0), r(1,1), r(1,2)], Box).
    Box = bbox(1,0,1,2).

measure_bbox_size/3 returns the width and height of the bounding box.

    ?- measure_bbox_size([r(0,0), r(2,3)], W, H).
    W = 4, H = 3.


Perimeter

measure_perimeter/2 counts the total exposed 4-connected edge faces. Each cell
can contribute 0 to 4 exposed edges depending on how many of its 4 neighbors
are outside the region.

    ?- measure_perimeter([r(0,0)], P).
    P = 4.          % single cell: all 4 faces exposed.

    ?- measure_perimeter([r(0,0),r(0,1),r(1,0),r(1,1)], P).
    P = 8.          % 2x2 square: each cell has 2 exposed faces.


Diameter

measure_diameter/2 returns the maximum Manhattan distance between any two cells
in the region.

    ?- measure_diameter([r(0,0), r(0,3)], D).
    D = 3.

    ?- measure_diameter([r(0,0)], D).
    D = 0.


Extent and Aspect

measure_extent/3 returns the area-to-bounding-box ratio as two integers N and D,
where N is the area and D is the bounding box area. A perfectly filled
rectangle has N = D.

    ?- measure_extent([r(0,0), r(1,0), r(1,1)], N, D).
    N = 3, D = 4.   % L-shape: 3 cells in a 2x2 bounding box.

measure_aspect/3 returns the aspect ratio as N/D where N = max(W,H) and
D = min(W,H). A square has N = D.

    ?- measure_aspect([r(1,0), r(1,1), r(1,2)], N, D).
    N = 3, D = 1.   % hline: 3 wide, 1 tall.


Row and Column Spans

measure_row_span/2 and measure_col_span/2 count the number of distinct rows and columns
occupied by the region.

    ?- measure_row_span([r(0,1), r(1,1), r(2,1)], S).
    S = 3.          % vline occupies 3 distinct rows.

    ?- measure_col_span([r(0,1), r(1,1), r(2,1)], S).
    S = 1.          % vline occupies 1 distinct column.


Centroid and Radius

measure_centroid/3 returns the integer floor centroid (average row and column).

    ?- measure_centroid([r(0,0), r(0,2), r(2,0), r(2,2)], R, C).
    R = 1, C = 1.

measure_radius/2 returns the maximum Chebyshev distance from the centroid to any
cell in the region. The Chebyshev distance is max(|dR|, |dC|).

    ?- measure_radius([r(0,0), r(0,1), r(0,2)], Rad).
    Rad = 1.


Interior and Border

measure_interior_count/2 counts cells whose 4 neighbors are all inside the region.
measure_border_count/2 counts cells with at least one 4-neighbor outside. These
two counts always sum to the area.

    Sq3 = [r(0,0),r(0,1),r(0,2),r(1,0),r(1,1),r(1,2),r(2,0),r(2,1),r(2,2)].
    ?- measure_interior_count(Sq3, N).
    N = 1.     % only the center cell (1,1) has all 4 neighbors in the region.

    ?- measure_border_count(Sq3, N).
    N = 8.     % the 8 surrounding cells are border cells.


Color Count

measure_color_count/2 counts the number of distinct integer values present in a
grid (list-of-lists).

    ?- measure_color_count([[0,1,2],[0,1,2]], N).
    N = 3.

    ?- measure_color_count([[0,0],[0,0]], N).
    N = 1.


Exported Predicates

measure_area/2          - cell count
measure_bbox/2          - tight bounding box
measure_bbox_size/3     - bounding box width and height
measure_perimeter/2     - exposed 4-edge face count
measure_diameter/2      - maximum Manhattan distance between any two cells
measure_extent/3        - area/bbox_area as N/D
measure_aspect/3        - max(W,H)/min(W,H) as N/D
measure_row_span/2      - distinct row count
measure_col_span/2      - distinct column count
measure_centroid/3      - integer floor centroid
measure_radius/2        - maximum Chebyshev distance from centroid
measure_interior_count/2 - cells with all 4 neighbors inside region
measure_border_count/2  - cells with at least one neighbor outside region
measure_color_count/2   - distinct color values in a grid
