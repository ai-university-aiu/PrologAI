Chapter 141: Layout Pack (lt_*)

The layout pack analyzes the collective spatial arrangement of a list of
obj(Color, Cells) terms. All centroid computations use integer truncation.

1. Global Bounding Box

layout_global_bbox(+Objs, -R1, -C1, -R2, -C2): bounding box over ALL cells.

  ?- layout_global_bbox([obj(a,[r(0,0)]),obj(b,[r(3,5)])], R1, C1, R2, C2).
  R1=0, C1=0, R2=3, C2=5.

layout_bbox_area(+Objs, -Area): (R2-R1+1) * (C2-C1+1).

2. Row and Column Range

layout_row_range(+Objs, -MinR, -MaxR): range of topmost rows.
layout_col_range(+Objs, -MinC, -MaxC): range of leftmost columns.
layout_row_count(+Objs, -N): number of distinct topmost rows.
layout_col_count(+Objs, -N): number of distinct leftmost columns.

3. Line Detection

layout_all_same_row(+Objs): succeed iff all objects share the same topmost row.
layout_all_same_col(+Objs): succeed iff all objects share the same leftmost column.

  ?- layout_all_same_row([obj(a,[r(0,0)]),obj(b,[r(0,3)]),obj(c,[r(0,7)])]).
  true.

4. Centroid Aggregation

layout_centroid_of_all(+Objs, -R, -C): integer-truncated centroid of all centroids.

5. Grid Arrangement

layout_is_grid(+Objs, -Rows, -Cols): objects' centroids form an R x C grid.
Every (row, col) combination must be occupied; |Objs| = Rows * Cols.

  ?- layout_is_grid([obj(a,[r(0,0)]),obj(b,[r(0,2)]),
                 obj(c,[r(2,0)]),obj(d,[r(2,2)])], R, C).
  R=2, C=2.

6. Diagonal Detection

layout_is_diagonal_dr(+Objs): all centroids on same downward-right diagonal (R-C constant).
layout_is_diagonal_dl(+Objs): all centroids on same downward-left diagonal (R+C constant).

  ?- layout_is_diagonal_dr([obj(a,[r(0,0)]),obj(b,[r(1,1)]),obj(c,[r(2,2)])]).
  true.

7. Uniform Spacing

layout_gap_h(+Objs, -Gap): uniform horizontal gap between centroid-sorted objects.
layout_gap_v(+Objs, -Gap): uniform vertical gap. Fails if gaps are non-uniform.

  ?- layout_gap_h([obj(a,[r(0,0)]),obj(b,[r(0,3)]),obj(c,[r(0,6)])], Gap).
  Gap = 3.
