155.1 Overview

The objfilter pack narrows a list of obj(Color, Cells) terms down to
the objects that match a criterion. Given a scene (a list of objects),
you can select by color, by cell count, by shape, or by a custom test.

The pack has 14 exported predicates (of_* prefix) and 9 private helpers.
It has no cross-pack dependencies; shape tests are self-contained.

Layer: 173. No dependencies on other packs.

155.2 Color Filters

object_filter_by_color(+Objs, +Color, -Filtered)

Keeps objects whose color equals Color.

    object_filter_by_color(Scene, red, Reds).   % only red objects

object_filter_not_color(+Objs, +Color, -Filtered)

Keeps objects whose color is NOT Color.

    object_filter_not_color(Scene, black, Rest).  % everything except black

155.3 Size Filters

object_filter_exact_size(+Objs, +N, -Filtered)

Keeps objects with exactly N cells.

    object_filter_exact_size(Scene, 1, Singles).  % single-cell objects

object_filter_min_size(+Objs, +N, -Filtered)

Keeps objects with at least N cells.

    object_filter_min_size(Scene, 4, Big).  % objects with 4 or more cells

object_filter_max_size(+Objs, +N, -Filtered)

Keeps objects with at most N cells.

    object_filter_max_size(Scene, 3, Small).  % objects with 3 or fewer cells

155.4 Shape Filters

object_filter_is_rect(+Objs, -Filtered)

Keeps objects that fill their bounding box (no holes). Solid rectangles,
horizontal bars, vertical bars, and single cells all qualify.

    object_filter_is_rect(Scene, Rects).

object_filter_is_hline(+Objs, -Filtered)

Keeps objects where all cells lie in a single row (bounding box height = 1).

    object_filter_is_hline(Scene, Hlines).

object_filter_is_vline(+Objs, -Filtered)

Keeps objects where all cells lie in a single column (bounding box width = 1).

    object_filter_is_vline(Scene, Vlines).

object_filter_is_single(+Objs, -Filtered)

Keeps single-cell objects (exactly 1 cell).

    object_filter_is_single(Scene, Dots).

object_filter_is_hollow(+Objs, -Filtered)

Keeps objects with at least one hole inside their bounding box (cell count
is less than the bounding box area). L-shapes and frames are hollow.

    object_filter_is_hollow(Scene, Hollow).

155.5 Aggregate Selection

object_filter_largest(+Objs, -Obj)

Returns the object with the most cells. On ties, returns the first in the
list. Fails when Objs is empty.

    object_filter_largest(Scene, Biggest).

object_filter_smallest(+Objs, -Obj)

Returns the object with the fewest cells. On ties, returns the first in
the list. Fails when Objs is empty.

    object_filter_smallest(Scene, Smallest).

155.6 Generic Meta-Predicates

object_filter_filter(+Objs, :Goal, -Filtered)

Keeps objects where call(Goal, Obj) succeeds. Wraps include/3.

    is_square_bbox(obj(_, Cells)) :-
        findall(R, member(r(R,_), Cells), Rs),
        findall(C, member(r(_,C), Cells), Cs),
        min_list(Rs, MinR), max_list(Rs, MaxR),
        min_list(Cs, MinC), max_list(Cs, MaxC),
        MaxR - MinR =:= MaxC - MinC.

    object_filter_filter(Scene, is_square_bbox, Squares).

object_filter_partition(+Objs, :Goal, -In, -Out)

Splits Objs into two lists: In (where Goal succeeds) and Out (where Goal
fails). Wraps partition/4.

    object_filter_partition(Scene, is_square_bbox, Squares, NonSquares).
