Chapter 212: gridshift - Grid Shifting and Cyclic Rolling (gsh_*, Layer 230)

Overview

The gridshift pack provides fourteen gsh_* predicates for shifting and rolling
symbolic grids. It covers linear shifts (bg fill), toroidal rolls, single-row
and single-column rolling, color-specific shifting, and full-grid offsetting.

Pack: gridshift
Module: gridshift
Prefix: gsh_
Layer: 230
Dependencies: library(lists)


Linear Shifts (with background fill)

gridshift_shift_right(+Grid, +N, +BgColor, -Result)
Shift each row right by N; leftmost N cells become BgColor.

    G = [[a,b,c],[d,e,f],[g,h,i]].
    gridshift_shift_right(G, 1, z, R).
    % R = [[z,a,b],[z,d,e],[z,g,h]]

gridshift_shift_left(+Grid, +N, +BgColor, -Result)
Shift each row left by N; rightmost N cells become BgColor.

gridshift_shift_down(+Grid, +N, +BgColor, -Result)
Shift grid down by N rows; N BgColor rows appear at the top.

    gridshift_shift_down(G, 1, z, R).
    % R = [[z,z,z],[a,b,c],[d,e,f]]

gridshift_shift_up(+Grid, +N, +BgColor, -Result)
Shift grid up by N rows; N BgColor rows appear at the bottom.


Cyclic Rolls (toroidal, no data loss)

gridshift_roll_right(+Grid, +N, -Result)
Cyclically roll each row right by N; last N cells wrap to front.

    gridshift_roll_right(G, 1, R).
    % R = [[c,a,b],[f,d,e],[i,g,h]]

gridshift_roll_left(+Grid, +N, -Result)
Cyclically roll each row left by N; first N cells wrap to back.

gridshift_roll_down(+Grid, +N, -Result)
Cyclically roll entire grid down by N rows; last N rows wrap to top.

    gridshift_roll_down(G, 1, R).
    % R = [[g,h,i],[a,b,c],[d,e,f]]

gridshift_roll_up(+Grid, +N, -Result)
Cyclically roll entire grid up by N rows; first N rows wrap to bottom.

Roll inverses: gridshift_roll_right(G, N, R1), gridshift_roll_left(R1, N, G) holds for any N.
Full rotation: gridshift_roll_right(G, W, G) and gridshift_roll_down(G, H, G) always hold.


Per-Row and Per-Column Operations

gridshift_roll_row(+Grid, +R, +N, -Result)
Cyclically roll row R right by N; all other rows are unchanged.

    gridshift_roll_row(G, 0, 1, R).
    % R = [[c,a,b],[d,e,f],[g,h,i]]

gridshift_roll_col(+Grid, +C, +N, -Result)
Cyclically roll column C down by N; all other columns are unchanged.
Implemented via transpose.

gridshift_shift_row(+Grid, +R, +N, +BgColor, -Result)
Shift row R by N (positive = right, negative = left); vacated cells = BgColor.
Other rows are unchanged.

gridshift_shift_col(+Grid, +C, +N, +BgColor, -Result)
Shift column C by N (positive = down, negative = up); vacated cells = BgColor.
Implemented via transpose.


Color-Specific and Full-Grid Offset

gridshift_shift_color(+Grid, +Color, +DR, +DC, +BgColor, -Result)
Shift all cells of Color by (DR, DC). Cells that move out-of-bounds disappear.
Vacated positions become BgColor. Non-Color non-BgColor cells are unchanged.

    G = [[x,b,b],[b,b,b],[b,b,y]].
    gridshift_shift_color(G, x, 0, 1, b, R).
    % R = [[b,x,b],[b,b,b],[b,b,y]]  (x moved right; y unchanged)

gridshift_offset(+Grid, +DR, +DC, +BgColor, -Result)
Shift the entire grid by (DR, DC): Result[R][C] = Grid[R-DR][C-DC] if in bounds,
else BgColor. Equivalent to gridshift_shift_right(N=DC) then gridshift_shift_down(N=DR) for
non-negative offsets.

    gridshift_offset(G, 1, 1, b, R).
    % R = [[b,b,b],[b,x,b],[b,b,b]]  (everything shifted right and down)

Equivalence: gridshift_shift_right(G, N, Bg, R) equals gridshift_offset(G, 0, N, Bg, R).
