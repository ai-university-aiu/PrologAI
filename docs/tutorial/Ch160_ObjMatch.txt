160.1 Overview

The objmatch pack finds correspondences between two obj(Color, Cells) term
lists. It answers the question: which object in list A corresponds to which
object in list B?

The pack has 14 exported predicates (mx_* prefix) and no cross-pack
dependencies.

Layer: 178. No dependencies on other packs.

160.2 Matching by Color

objmatch_by_color(+List1, +List2, -Pairs)

Each Obj1 in List1 is paired with every Obj2 in List2 that shares the same
color. All matches are returned (multiple matches per object are possible).

    objmatch_by_color(
        [obj(r,[r(0,0)]), obj(b,[r(1,0)])],
        [obj(r,[r(5,5)]), obj(b,[r(6,6)])],
        Pairs).
    % Pairs = [obj(r,[r(0,0)])-obj(r,[r(5,5)]),
    %          obj(b,[r(1,0)])-obj(b,[r(6,6)])]

160.3 Matching by Size

objmatch_by_size(+List1, +List2, -Pairs)

Each Obj1 is paired with each Obj2 that has the same cell count (size). Color
is ignored.

    objmatch_by_size(
        [obj(r,[r(0,0)])],
        [obj(b,[r(0,0)]), obj(g,[r(1,1)])],
        Pairs).
    % Pairs = [obj(r,[r(0,0)])-obj(b,[r(0,0)]),
    %          obj(r,[r(0,0)])-obj(g,[r(1,1)])]

160.4 Matching by Form

objmatch_by_form(+List1, +List2, -Pairs)

Each Obj1 is paired with each Obj2 whose normalized shape equals Obj1's. The
normalized shape is the sorted cell list after translating the bounding box
top-left to r(0,0). Color and position are both ignored.

    Lshape = obj(p, [r(0,0),r(1,0),r(1,1)]),
    LshapeFar = obj(y, [r(3,4),r(4,4),r(4,5)]),
    objmatch_by_form([Lshape], [LshapeFar], Pairs).
    % Pairs = [Lshape-LshapeFar]

160.5 Matching by Nearest Centroid (Greedy)

objmatch_by_nearest(+List1, +List2, -Pairs)

Greedy nearest-centroid matching. Each Obj1 in List1 claims the nearest
remaining Obj2 by squared Euclidean centroid distance.

    A = obj(r,[r(0,0)]),   % centroid r(0,0)
    B = obj(b,[r(2,2)]),   % centroid r(2,2)
    C = obj(g,[r(5,5)]),   % centroid r(5,5)
    D = obj(y,[r(1,1)]),   % centroid r(1,1)
    objmatch_by_nearest([A, C], [B, D], Pairs).
    % A (at 0,0) nearest to D (at 1,1, dist^2=2) vs B (at 2,2, dist^2=8)
    % C (at 5,5) nearest to B (at 2,2, remaining)
    % Pairs = [A-D, C-B]

160.6 Extracting Unmatched Objects

objmatch_unmatched(+Pairs, +List1, +List2, -Unmatched1, -Unmatched2)

Returns the objects from each list that do not appear in any pair.

    R = obj(r,[r(0,0)]), B = obj(b,[r(0,0)]), G = obj(g,[r(0,0)]),
    objmatch_unmatched([R-B], [R, G], [B, G], Um1, Um2).
    % Um1 = [G], Um2 = [G]

160.7 Filtering Pairs by Color Change

objmatch_filter_changed_color(+Pairs, -Changed)

Keeps only pairs where the two colors differ.

objmatch_filter_same_color(+Pairs, -Same)

Keeps only pairs where the two colors are identical.

160.8 Computing Deltas

objmatch_color_deltas(+Pairs, -Deltas)

Maps each O1-O2 pair to a c1-c2 atom pair showing the color change.

    objmatch_color_deltas([obj(r,[r(0,0)])-obj(b,[r(0,0)])], D).
    % D = [r-b]

objmatch_pos_deltas(+Pairs, -Deltas)

Maps each pair to a dr(DR,DC) term showing the centroid displacement.

    objmatch_pos_deltas([obj(r,[r(0,0)])-obj(b,[r(3,4)])], D).
    % D = [dr(3,4)]

objmatch_size_deltas(+Pairs, -Deltas)

Maps each pair to an integer N2-N1 showing the size change.

    objmatch_size_deltas([obj(r,[r(0,0)])-obj(b,[r(0,0),r(0,1)])], D).
    % D = [1]

160.9 Testing Uniformity of Deltas

objmatch_all_same_color_delta(+Pairs)

Succeeds iff all pairs share the same c1-c2 color change (trivially true for
the empty list).

objmatch_all_same_pos_delta(+Pairs)

Succeeds iff all pairs share the same dr(DR,DC) positional delta.

objmatch_all_same_size_delta(+Pairs)

Succeeds iff all pairs share the same integer size delta.

160.10 Positional Zip

objmatch_zip(+List1, +List2, -Pairs)

Index-wise pairing: element I of List1 with element I of List2.

    objmatch_zip([A, B, C], [X, Y, Z], Pairs).
    % Pairs = [A-X, B-Y, C-Z]

The lists must have equal length; the predicate fails if they differ.
