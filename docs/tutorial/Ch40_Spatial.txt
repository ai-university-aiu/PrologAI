Chapter 40: The Spatial Pack

The spatial pack (module spatial, sp_* prefix, Layer 58) gives PrologAI
fourteen predicates for spatial reasoning about positions and regions.

Where the measure pack describes the size of a region and the compare
pack describes what changed between two grids, the spatial pack answers
directional and positional questions: where is this cell relative to
that one, how far apart are they, and what are their neighbors?


40.1 Loading the Pack

To use the spatial pack in your code:

:- use_module(library(spatial)).


40.2 Direction Between Cells

spatial_direction(Cell1, Cell2, Dir) gives the cardinal direction from Cell1
to Cell2.

Dir is one of: north, south, east, west, northeast, northwest,
southeast, southwest, or same.

Example:

spatial_direction(r(3,2), r(1,2), Dir).
% Dir = north

spatial_direction(r(1,3), r(3,1), Dir).
% Dir = southwest


40.3 Distance Metrics

spatial_distance_manhattan(Cell1, Cell2, D) computes D = |R2-R1| + |C2-C1|.

This is the shortest path distance when only horizontal and vertical
moves are allowed.

spatial_distance_chebyshev(Cell1, Cell2, D) computes D = max(|R2-R1|, |C2-C1|).

This is the shortest path distance when diagonal moves are also allowed.

Example:

spatial_distance_manhattan(r(0,0), r(2,2), D).
% D = 4

spatial_distance_chebyshev(r(0,0), r(2,2), D).
% D = 2


40.4 Neighbor Enumeration

spatial_neighbors4(Cell, Rows-Cols, Neighbors) returns the list of in-bounds
4-connected neighbors (up, down, left, right).

spatial_neighbors8(Cell, Rows-Cols, Neighbors) returns all 8-connected
in-bounds neighbors (including diagonals).

Example:

spatial_neighbors4(r(1,1), 3-3, N).
% N = [r(0,1), r(2,1), r(1,0), r(1,2)]  (order may vary)


40.5 Adjacency Tests

spatial_adjacent4(Cell1, Cell2) succeeds when Manhattan distance is exactly 1.

spatial_adjacent8(Cell1, Cell2) succeeds when Chebyshev distance is exactly 1.


40.6 Containment Tests

spatial_bbox_contains(bbox(R1,C1,R2,C2), Cell) succeeds when Cell is inside
the bounding box (inclusive on all edges).

spatial_in_region(Cell, Region) succeeds when Cell is a member of a region
list. Deterministic: the cut removes the choicepoint.


40.7 Region Band Filtering

spatial_row_between(Region, MinR, MaxR, Filtered) keeps cells with row index
in [MinR, MaxR].

spatial_col_between(Region, MinC, MaxC, Filtered) keeps cells with column
index in [MinC, MaxC].

Example:

Region = [r(0,0), r(1,0), r(2,0), r(3,0)],
spatial_row_between(Region, 1, 2, Filtered).
% Filtered = [r(1,0), r(2,0)]


40.8 Nearest and Farthest

spatial_closest(Ref, Region, Closest) returns the cell in Region with the
smallest Manhattan distance from Ref. On ties, the first cell in the
list is returned.

spatial_farthest(Ref, Region, Farthest) returns the cell with the largest
Manhattan distance.

Example:

spatial_closest(r(0,0), [r(5,5), r(1,1), r(3,3)], Closest).
% Closest = r(1,1)


40.9 Integer Centroid

spatial_centroid(Region, Row, Col) computes the integer centroid of a region:
Row = sum_of_row_indices // count, Col = sum_of_col_indices // count.

The result is truncated (not rounded), consistent with the measure pack.
