Chapter 181: gridnbr Pack - Grid Neighbor Analysis (gn_*, Layer 199)


Overview

The gridnbr pack analyzes the neighborhood of cells in raw grid format
(list-of-lists of color atoms). It provides predicates for reading neighbor
values, counting specific colors in neighborhoods, testing neighborhood
conditions, applying morphological dilation and erosion, and running
Conway-style cellular automaton steps.

Layer 199. Pack prefix: gn_*. Work Package 220.


Exported Predicates


gridnbr_nbr4(+Grid, +R, +C, -Vals)

Vals is the list of cell values at the 4-connected (cardinal) neighbors of
(R,C): up, down, left, right. Out-of-bounds positions are omitted. For a
corner cell the list has 2 elements; for an edge cell 3; for an interior cell 4.

Example:

% Center of 3x3 grid has 4 neighbors
gridnbr_nbr4([[r,r,r],[r,b,r],[r,r,r]], 1, 1, Vals).
% Vals = [r, r, r, r]  (up, down, left, right)

% Corner has 2 neighbors
gridnbr_nbr4([[r,r,r],[r,b,r],[r,r,r]], 0, 0, Vals).
% length(Vals, 2)


gridnbr_nbr8(+Grid, +R, +C, -Vals)

Like gridnbr_nbr4 but includes the 4 diagonal neighbors as well. List length is
3, 5, or 8 depending on position.


gridnbr_count4(+Grid, +R, +C, +Color, -N)

N is the count of 4-connected neighbors of (R,C) whose value equals Color.

Example:

gridnbr_count4([[r,r,r],[r,b,r],[r,r,r]], 1, 1, r, N).
% N = 4  (all 4 neighbors are r)


gridnbr_count8(+Grid, +R, +C, +Color, -N)

Like gridnbr_count4 but counts among all 8 neighbors.


gridnbr_count4_grid(+Grid, +Color, -CountGrid)

CountGrid is a raw grid of the same dimensions where each cell holds the
count of Color in its 4-connected neighbors.

Example:

gridnbr_count4_grid([[r,r],[r,r]], b, CG).
% CG = [[0,0],[0,0]]


gridnbr_count8_grid(+Grid, +Color, -CountGrid)

Like gridnbr_count4_grid but uses 8-connected neighborhoods.


gridnbr_any4(+Grid, +R, +C, +Color)

Succeed (deterministically) if at least one 4-connected neighbor of (R,C)
has value Color.

Example:

gridnbr_any4([[r,r,r],[r,b,r],[r,r,r]], 1, 1, r).  % succeeds
\+ gridnbr_any4([[r,r,r],[r,b,r],[r,r,r]], 1, 1, b).  % no b neighbor


gridnbr_all4(+Grid, +R, +C, +Color)

Succeed if every 4-connected neighbor of (R,C) has value Color. Fails if
the cell has no 4-neighbors (1x1 grid).

Example:

% Ring grid: r at center, b everywhere else
gridnbr_all4([[b,b,b],[b,r,b],[b,b,b]], 1, 1, b).  % all 4 neighbors are b


gridnbr_mark_border(+Grid, +Bg, -MarkedGrid)

MarkedGrid is Grid with every non-background cell that has at least one Bg
4-neighbor replaced by the atom border. All other cells are left unchanged.
A "border cell" is a foreground cell that touches the background.

Example:

gridnbr_mark_border([[r,r,r],[r,b,r],[r,r,r]], r, MG).
% MG = [[r,r,r],[r,border,r],[r,r,r]]  (b at center touches r)


gridnbr_mark_isolated(+Grid, +Bg, -MarkedGrid)

MarkedGrid is Grid with every non-background cell that has NO same-color
4-neighbor replaced by the atom isolated. A "connected" foreground cell
touches another cell of the same color.

Example:

% Two isolated b cells that don't touch each other
gridnbr_mark_isolated([[r,r,r],[b,r,b],[r,r,r]], r, MG).
% MG = [[r,r,r],[isolated,r,isolated],[r,r,r]]


gridnbr_expand_color(+Grid, +Color, -Expanded)

Expanded is Grid after morphological dilation of Color: every background
cell with at least one Color 4-neighbor is set to Color. The background is
the most frequent color.

Example:

gridnbr_expand_color([[r,r,r],[r,b,r],[r,r,r]], b, E).
% E = [[r,r,r],[b,b,b],[r,r,r]]  (left and right of b are filled)


gridnbr_shrink_color(+Grid, +Color, -Shrunk)

Shrunk is Grid after morphological erosion of Color: every Color cell with
at least one non-Color 4-neighbor is set to the background color.

Example:

% 5x5 grid with 3x3 b block: outer ring of b eroded away
gridnbr_shrink_color(G5x5, b, Shrunk).
% Only the center cell remains b; outer b cells become r


gridnbr_majority_nbr4(+Grid, +R, +C, -Maj)

Maj is the most frequently occurring color among the 4-connected neighbors.
Ties broken by standard term order (smallest atom wins). Fails for cells
with no 4-neighbors.


gridnbr_conway_step(+Grid, +Color, +N, -NewGrid)

NewGrid is Grid after one cellular automaton step: every Color cell with
exactly N Color 4-neighbors is replaced by the background color. All other
cells are unchanged. The background is the most frequent color.

Example:

% Isolated b (0 b-neighbors) is killed by rule N=0
gridnbr_conway_step([[r,r,r],[r,b,r],[r,r,r]], b, 0, NG).
% NG = [[r,r,r],[r,r,r],[r,r,r]]


Typical Use Cases

Detecting isolated foreground cells (noise removal):
  gridnbr_mark_isolated(Grid, Bg, MG)

Detecting object borders:
  gridnbr_mark_border(Grid, Bg, MG)

Morphological dilation (grow objects by one layer):
  gridnbr_expand_color(Grid, Color, Expanded)

Morphological erosion (shrink objects by one layer):
  gridnbr_shrink_color(Grid, Color, Shrunk)

Counting-based grid transformation:
  gridnbr_count4_grid(Grid, TargetColor, CountGrid)

Summary

cells, and applying morphological grow and shrink operations to raw grids.
