Chapter 261: Wave 43 - frame_stamp (WP-301, Layer 276)

frame_stamp solves ARC-AGI-2 task 247ef758. A bordered room occupies the
right portion of the grid (columns 4 and beyond). The room's outer frame has
a dominant color (BdrBG). Anomalous values in the top row of the frame encode
a stamp column per color (col K encodes color C). Anomalous values in the
rightmost column of the frame encode a stamp row per color (row R encodes
color C). Colored shapes occupy the left side (columns 0-3) of the grid.

For each color C that has both a top-row column anomaly (giving col K) and a
right-column row anomaly (giving row R), and also has a left-side shape: the
shape is stamped inside the room centered at (R, K). Multiple placements per
color are supported when multiple anomalies exist. Stamps are applied in
descending cell-count order so smaller shapes overwrite larger ones where they
overlap. Left-side shapes that were stamped are erased to BG. Left-side shapes
without any frame anomalies are preserved unchanged.
