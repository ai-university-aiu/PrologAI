Chapter 209: Grid Patch Operations (gridpatch, gpt_*)

Layer 227 - Work Package 248

The gridpatch pack provides fourteen predicates for working with sub-grid regions
(patches): extracting, placing, overlaying, searching, tiling, scattering, and
clearing. A patch is itself a grid (list of rows of color atoms).


1. Extracting and Placing

gridpatch_extract/6 copies a rectangular region from a grid:

  G = [[x,b,x,b],[b,x,b,x],[x,b,x,b],[b,x,b,x]].
  gridpatch_extract(G, 0, 0, 1, 1, P).  % P = [[x,b],[b,x]]

gridpatch_place/5 pastes a patch at a position, overwriting cells:

  gridpatch_place(G, [[m,m],[m,m]], 1, 1, R).
  % R row 1 = [b,m,m,x], row 2 = [x,m,m,b]


2. Overlay with Transparency

gridpatch_overlay/6 pastes a patch but skips cells with a transparency color:

  Patch = [[b,m],[m,b]].
  gridpatch_overlay(G, Patch, 0, 0, b, R).
  % b cells in Patch are transparent: Grid cells shine through
  % m cells in Patch overwrite the Grid


3. Searching for Patches

gridpatch_match_at/4 checks if a patch matches at a specific position:

  gridpatch_match_at(G, [[x,b],[b,x]], 0, 0).  % succeeds
  gridpatch_match_at(G, [[x,x],[x,x]], 0, 0).  % fails

gridpatch_find_all/3 returns all matching positions:

  gridpatch_find_all(G, [[x,b],[b,x]], Pos).
  % Pos = [0-0, 0-2, 1-1, 2-0, 2-2]  (5 positions in 4x4 checker)

gridpatch_count/3 counts them:

  gridpatch_count(G, [[x,b],[b,x]], N).  % N = 5

gridpatch_replace_first/4 replaces the first (top-left) match:

  gridpatch_replace_first(G, [[x,b],[b,x]], [[z,z],[z,z]], R).
  % replaces the match at (0,0)


4. Tiling and Scattering

gridpatch_tile_fill/4 builds an H x W grid by repeating a tile:

  gridpatch_tile_fill([[x,b],[b,x]], 4, 4, G).  % G = checker grid

gridpatch_scatter/4 places a patch at each of a list of positions:

  gridpatch_scatter(G, [[m]], [0-0, 2-2, 3-3], R).
  % places [[m]] at three positions, left-to-right


5. Dimensions and Equality

gridpatch_h/2, gridpatch_w/2, and gridpatch_size/3 return patch dimensions:

  gridpatch_h([[x,b],[b,x]], H).  % H = 2
  gridpatch_w([[x,b],[b,x]], W).  % W = 2
  gridpatch_size([[x,b],[b,x]], H, W).  % H = 2, W = 2

gridpatch_eq/2 checks if two patches are identical:

  gridpatch_eq([[x,b],[b,x]], [[x,b],[b,x]]).  % succeeds


6. Inpainting

gridpatch_inpaint/7 clears a rectangular region to a fill color:

  gridpatch_inpaint(G, 1, 1, 2, 2, b, R).  % sets rows 1-2, cols 1-2 to b


7. Predicate Reference

  gridpatch_extract/6       - extract sub-grid region
  gridpatch_place/5         - paste patch at position (overwrite)
  gridpatch_overlay/6       - paste patch with transparency
  gridpatch_match_at/4      - check match at position
  gridpatch_find_all/3      - all positions where patch matches
  gridpatch_count/3         - count matching positions
  gridpatch_replace_first/4 - replace first matching occurrence
  gridpatch_scatter/4       - place patch at each listed position
  gridpatch_tile_fill/4     - build H x W grid by tiling patch
  gridpatch_h/2             - height (row count) of patch
  gridpatch_w/2             - width (column count) of patch
  gridpatch_size/3          - height and width of patch
  gridpatch_eq/2            - patch equality check
  gridpatch_inpaint/7       - fill rectangular region with color
