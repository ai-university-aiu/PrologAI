Chapter 175: multicolor Pack - Multi-Color Scene Analysis (mc_*, Layer 193)


Overview

The multicolor pack queries and partitions obj(Color,Cells) scenes by color
frequency and membership. It answers questions like: which color appears most
often? which colors appear exactly once? what are all pairs of colors in the
scene? does any color hold a majority?

Layer 193. Pack prefix: mc_*. Work Package 214.


Exported Predicates


multi_color_color_counts(+Scene, -Counts)

Sorted Color-N pairs, one per distinct color.

Example:

multi_color_color_counts([obj(r,[r(0,0)]), obj(b,[r(0,1)]), obj(r,[r(1,0)])], Counts).
% Counts = [b-1, r-2]


multi_color_count_of_color(+Scene, +Color, -N)

Object count for a given color. N=0 if Color not present.

Example:

multi_color_count_of_color([obj(r,[r(0,0)]), obj(r,[r(0,1)])], r, N).
% N = 2


multi_color_color_present(+Scene, +Color)

Succeed if Color appears at least once in Scene.


multi_color_dominant_color(+Scene, -Color)

Color with the most objects. Ties broken by standard Prolog term order.

Example:

multi_color_dominant_color([obj(r,[r(0,0)]), obj(r,[r(0,1)]), obj(b,[r(1,0)])], Color).
% Color = r


multi_color_rarest_color(+Scene, -Color)

Color with the fewest objects. Ties broken by term order.


multi_color_singleton_colors(+Scene, -Colors)

All colors that appear in exactly one object.

Example:

multi_color_singleton_colors([obj(r,[r(0,0)]), obj(b,[r(0,1)]), obj(r,[r(1,0)])], Colors).
% Colors = [b]


multi_color_unique_color(+Scene, -Color)

Backtrackable: each color with count 1.


multi_color_color_freq_rank(+Scene, +N, -Color)

The Nth most frequent color (1 = most frequent). Fails if N exceeds the number
of distinct colors.

Example:

multi_color_color_freq_rank(Scene, 1, MostFrequent).
multi_color_color_freq_rank(Scene, 2, SecondMostFrequent).


multi_color_color_partition(+Scene, -Groups)

Color-Objs pairs, one per distinct color. Objs is the list of all objects
of that color.

Example:

multi_color_color_partition([obj(r,[r(0,0)]), obj(b,[r(0,1)])], Groups).
% Groups = [b-[obj(b,[r(0,1)])], r-[obj(r,[r(0,0)])]]


multi_color_pair_colors(+Scene, -Pairs)

All unordered color pairs as C1-C2 with C1 @< C2.

Example:

multi_color_pair_colors([obj(r,[r(0,0)]), obj(b,[r(0,1)]), obj(g,[r(1,0)])], Pairs).
% Pairs = [b-g, b-r, g-r]


multi_color_equal_count_pairs(+Scene, -Pairs)

Color pairs that have equal object count.

Example:

multi_color_equal_count_pairs([obj(r,[r(0,0)]), obj(b,[r(0,1)])], Pairs).
% Pairs = [b-r]  (both have count 1)


multi_color_majority_color(+Scene, -Color)

Color whose object count exceeds half the total scene size. Fails if none exists.

Example:

multi_color_majority_color([obj(r,[r(0,0)]), obj(r,[r(0,1)]), obj(r,[r(1,0)]), obj(b,[r(1,1)])], Color).
% Color = r  (3 out of 4 objects = 75%)


multi_color_n_distinct_colors(+Scene, -N)

Number of distinct colors in the scene.


multi_color_color_index(+Scene, +Color, -Index)

1-based frequency rank for a given color. The most frequent color has index 1.

Summary

The multicolor pack provides the color-frequency layer for multi-object scene
analysis. Combined with scene (for object discovery), colortable (for mapping),
and transformgen (for candidate generation), it supports ARC-AGI-2 tasks that
require reasoning about color distribution across a scene.
