Chapter 62: Sorting, Ranking, and Ordering - The Sort Pack

This chapter covers the sort pack (so_* predicates, Layer 80), which provides
14 predicates for summing, counting, sorting, and ranking operations on grids.
The sort pack answers questions like: which row has the most of this color?
Sort the columns so the emptiest come first.

62.1 Loading the Pack

:- use_module(library(sort)).

62.2 Row and Column Sums

sort_row_sums(+Grid, -Sums)
  Sums is a list with one integer per row: the sum of all cell values in that row.
  Values must be numeric.

Example:
  ?- sort_row_sums([[1,2],[3,4]], S).
  S = [3,7].

sort_col_sums(+Grid, -Sums)
  Sums is a list with one integer per column: the sum of all cell values in that column.

Example:
  ?- sort_col_sums([[1,2],[3,4]], S).
  S = [4,6].

62.3 Value Counting per Row and Column

sort_row_count(+Grid, +Val, -Counts)
  Counts is a list with one integer per row: the number of cells equal to Val.

Example:
  ?- sort_row_count([[1,0,1],[0,0,1]], 1, C).
  C = [2,1].

sort_col_count(+Grid, +Val, -Counts)
  Counts is a list with one integer per column: the number of cells equal to Val.

Example:
  ?- sort_col_count([[1,0],[1,0],[0,1]], 1, C).
  C = [2,1].

62.4 Sorting Rows

sort_sort_rows_asc(+Grid, +Val, -Sorted)
  Sorted has the same rows as Grid, ordered ascending by count of Val.
  Rows with fewer Val cells come first. Ties preserve original row order.

Example:
  ?- sort_sort_rows_asc([[1,1,0],[0,0,0],[1,0,0]], 1, S).
  S = [[0,0,0],[1,0,0],[1,1,0]].

sort_sort_rows_desc(+Grid, +Val, -Sorted)
  Sorted has the same rows as Grid, ordered descending by count of Val.
  Rows with more Val cells come first.

Example:
  ?- sort_sort_rows_desc([[0,0],[1,0],[1,1]], 1, S).
  S = [[1,1],[1,0],[0,0]].

62.5 Sorting Columns

sort_sort_cols_asc(+Grid, +Val, -Sorted)
  Sorted has the same columns as Grid, ordered ascending by count of Val.
  Columns with fewer Val cells come first.

sort_sort_cols_desc(+Grid, +Val, -Sorted)
  Sorted has the same columns as Grid, ordered descending by count of Val.

Note: Column sorting is implemented as: transpose, sort rows, transpose back.

62.6 Finding Extremal Rows and Columns

sort_max_row(+Grid, +Val, -R)
  R is the 0-based row index with the highest count of Val.
  On ties, returns the smallest index.

Example:
  ?- sort_max_row([[0,0],[1,0],[1,1]], 1, R).
  R = 2.

sort_min_row(+Grid, +Val, -R)
  R is the 0-based row index with the lowest count of Val.

sort_max_col(+Grid, +Val, -C)
  C is the 0-based column index with the highest count of Val.

sort_min_col(+Grid, +Val, -C)
  C is the 0-based column index with the lowest count of Val.

62.7 Value Ordering

sort_sorted_vals(+Grid, -Vals)
  Vals is all cell values from Grid sorted ascending, preserving duplicates.
  Useful for analyzing the distribution of values.

Example:
  ?- sort_sorted_vals([[3,1],[2,1]], V).
  V = [1,1,2,3].

sort_cell_rank(+Grid, +R, +C, -Rank)
  Rank is the 1-based position of Grid[R,C] among the distinct values of Grid
  in ascending order. The smallest distinct value has rank 1.

Example:
  ?- sort_cell_rank([[1,2],[3,4]], 1, 1, Rank).
  Rank = 4.

Example:
  ?- sort_cell_rank([[1,3],[5,7]], 0, 1, Rank).
  Rank = 2.  % 3 is the 2nd of [1,3,5,7]

62.8 Common Patterns

Pattern: Find the dominant color row:
  sort_max_row(Grid, 1, R),
  nth0(R, Grid, DominantRow).

Pattern: Bring the fullest rows to the top:
  sort_sort_rows_desc(Grid, ForegroundVal, Reordered).

Pattern: Compare row densities:
  sort_row_count(Grid, 1, Counts),
  msort(Counts, SortedCounts),
  last(SortedCounts, MaxCount).

Pattern: Get rank of each cell:
  length(Grid, NRows), Grid = [FR|_], length(FR, NCols),
  NRM1 is NRows-1, NCM1 is NCols-1,
  numlist(0, NRM1, Rs), numlist(0, NCM1, Cs),
  findall(R-C-Rank, (member(R, Rs), member(C, Cs), sort_cell_rank(Grid, R, C, Rank)), RankTriples).

62.9 Summary

The sort pack's 14 predicates organize into five groups:

Aggregation:  sort_row_sums/2, sort_col_sums/2, sort_row_count/3, sort_col_count/3
Row sort:     sort_sort_rows_asc/3, sort_sort_rows_desc/3
Column sort:  sort_sort_cols_asc/3, sort_sort_cols_desc/3
Extrema:      sort_max_row/3, sort_min_row/3, sort_max_col/3, sort_min_col/3
Ranking:      sort_sorted_vals/2, sort_cell_rank/4
