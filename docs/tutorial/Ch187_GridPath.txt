Chapter 187: gridpath Pack - BFS Pathfinding, Distance Maps, Flood-N, and Line-of-Sight (gpa_*, Layer 205)

Overview

The gridpath pack (Layer 205, gpa_* prefix) provides fourteen predicates for
grid pathfinding: BFS shortest-path finding, distance maps, flood expansion
to N steps, wavefront extraction, line-of-sight testing, straight-line segment
generation, and multi-region path discovery. ObsColor cells are impassable;
all movement is 4-connected (up, down, left, right).

Load the pack:

:- use_module(pack(gridpath/prolog/gridpath)).


1. Shortest-Path Operations

gridpath_shortest_path(+Grid, +R1, +C1, +R2, +C2, +ObsColor, -Path)

Finds the shortest 4-connected path from (R1,C1) to (R2,C2) avoiding
ObsColor cells. Path is a list of R-C coordinates including both endpoints.
Fails if either endpoint is ObsColor or if no path exists.

Example:

G = [[p,p,p],[p,w,p],[p,p,p]].
gridpath_shortest_path(G, 0, 0, 2, 2, w, Path).
% Path = [0-0, 0-1, 0-2, 1-2, 2-2]  or  [0-0, 1-0, 2-0, 2-1, 2-2]

gridpath_path_length(+Grid, +R1, +C1, +R2, +C2, +ObsColor, -N)

N is the number of steps (path length minus 1). If start = end, N = 0.

gridpath_reachable(+Grid, +R1, +C1, +R2, +C2, +ObsColor)

Succeeds if (R2,C2) is reachable from (R1,C1) via 4-connected non-ObsColor cells.

Example:

G = [[p,w,p],[p,w,p],[p,w,p]].
\+ gridpath_reachable(G, 0, 0, 0, 2, w).  % wall separates columns


2. Reachability and Distance Maps

gridpath_all_reachable(+Grid, +R, +C, +ObsColor, -Cells)

Cells is the list of all R-C positions reachable from (R,C) via 4-connected
non-ObsColor paths. Includes the seed cell itself.

gridpath_distance_map(+Grid, +R, +C, +ObsColor, -Map)

Map is a list of NR-NC-Dist triples: the BFS distance from (R,C) to every
reachable cell. The seed has Dist = 0; unreachable cells are omitted.

Example:

G = [[p,p,p],[p,p,p],[p,p,p]].
gridpath_distance_map(G, 0, 0, w, Map).
% Map includes 0-0-0, 0-1-1, 1-0-1, 0-2-2, 1-1-2, 2-0-2, 1-2-3, 2-1-3, 2-2-4


3. Flood and Wavefront Operations

gridpath_nearest(+Grid, +R, +C, +TargetColor, +ObsColor, -NR-NC)

NR-NC is the nearest reachable cell of TargetColor. Ties are broken by BFS
discovery order. Fails if no reachable TargetColor cell exists.

gridpath_flood_n(+Grid, +R, +C, +N, +ObsColor, -Cells)

Cells is the list of all R-C positions reachable within N steps from (R,C).
N = 0 returns just the seed cell.

Example:

G = [[p,p,p],[p,p,p],[p,p,p]].
gridpath_flood_n(G, 1, 1, 1, w, Cells).
% Cells = [1-1, 0-1, 2-1, 1-0, 1-2]  (center + 4 neighbors)

gridpath_wavefront(+Grid, +R, +C, +N, +ObsColor, -Cells)

Cells contains only the cells at exactly BFS distance N. Returns [] if no
cells exist at that distance (e.g., N is larger than the grid).

Example:

G = [[p,p,p],[p,p,p],[p,p,p]].
gridpath_wavefront(G, 1, 1, 2, w, Cells).
% Cells = [0-0, 0-2, 2-0, 2-2]  (4 corners)


4. Straight-Line Operations

gridpath_straight_h(+R, +C1, +C2, -Cells)

Cells is the list of R-C positions on row R from column C1 to column C2
inclusive. Direction follows C1-to-C2 order (left-to-right when C1 =< C2,
right-to-left when C1 > C2).

gridpath_straight_v(+C, +R1, +R2, -Cells)

Cells is the list of R-C positions on column C from row R1 to row R2 inclusive.

gridpath_between_h(+R, +C1, +C2, -Cells)

Cells is the strictly interior cells of the horizontal segment (endpoints
excluded), in ascending column order. Returns [] for adjacent cells.

gridpath_between_v(+R1, +R2, +C, -Cells)

Cells is the strictly interior cells of the vertical segment (endpoints
excluded), in ascending row order.

Examples:

gridpath_straight_h(0, 0, 4, Cells).
% Cells = [0-0, 0-1, 0-2, 0-3, 0-4]

gridpath_between_h(0, 0, 4, Cells).
% Cells = [0-1, 0-2, 0-3]

gridpath_between_v(0, 3, 1, Cells).
% Cells = [1-1, 2-1]


5. Line-of-Sight and Region Path

gridpath_line_of_sight(+Grid, +R1, +C1, +R2, +C2, +ObsColor)

Succeeds if (R1,C1) and (R2,C2) share the same row or column AND no ObsColor
cell lies strictly between them. Fails for diagonal pairs or if any intervening
cell is ObsColor.

Example:

G = [[p,p,p,p,p],[p,w,w,p,p],[p,p,p,p,p]].
gridpath_line_of_sight(G, 1, 0, 1, 4, w).  % fails: w at (1,1) and (1,2) block it
gridpath_line_of_sight(G, 0, 0, 0, 4, w).  % succeeds: row 0 is clear

gridpath_region_path(+Grid, +Color1, +Color2, +ObsColor, -Path)

Finds the shortest 4-connected path from any cell of Color1 to any cell of
Color2, avoiding ObsColor. Returns the globally shortest such path.
Fails if no Color1 or Color2 cells exist, or no path can be found.

Example:

G = [[r,p,p,p],[p,p,p,p],[p,p,p,p],[p,p,p,b]].
gridpath_region_path(G, r, b, w, Path).
% Path starts at 0-0 and ends at 3-3 with length 7


6. Summary of Predicates

gridpath_shortest_path/7  - BFS shortest 4-path avoiding ObsColor (with path list)
gridpath_path_length/7    - step count of shortest path
gridpath_reachable/6      - succeed if target reachable from seed
gridpath_all_reachable/5  - all cells reachable from seed
gridpath_distance_map/5   - BFS distance from seed to each reachable cell
gridpath_nearest/6        - nearest reachable cell of a given color
gridpath_flood_n/6        - all cells reachable within N steps
gridpath_wavefront/6      - cells at exactly N BFS steps
gridpath_straight_h/4     - cells on horizontal segment (inclusive)
gridpath_straight_v/4     - cells on vertical segment (inclusive)
gridpath_line_of_sight/6  - same row or column with no obstacle between
gridpath_between_h/4      - interior cells of horizontal segment (exclusive)
gridpath_between_v/4      - interior cells of vertical segment (exclusive)
gridpath_region_path/5    - shortest path between any Color1 and Color2 cell
